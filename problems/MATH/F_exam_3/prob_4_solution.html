PART 1 — Problem Analysis (no solving yet)

Interpret • Identify • Choose method

Restated Task

Compute the following definite integrals, stating a suitable method for each:

(a) ∫[0, √5] (1 + 2x + 3x² + 4x³ + 5x⁴) dx (b) ∫[0, 10] x / √(1 + x²) dx (c) ∫[-π/2, π/2] sin⁷(θ) dθ (d) ∫[1, 2] ln(t²) / t dt (e) ∫[0, π/4] 4 sin(2t) / cos(t) dt

Given

  • Integrands and limits for parts (a)–(e).

Unknowns

  • The exact values of each definite integral.

Core principles

  • Linearity of integrals (split sums, pull constants).
  • FTC: evaluate antiderivative at bounds.
  • Substitution: change variables to simplify.
  • Symmetry: odd functions cancel on symmetric limits.
  • Trig identities: simplify before integrating.

Candidate approaches

  • Direct antiderivatives (polynomials, basic trig).
  • \(u\)-substitution (recognize “function + derivative”).
  • Symmetry / identities (odd/even, trig reductions).

Best method per part

(a) Power rule term-by-term.   (b) \(u=1+x^2\).   (c) Odd symmetry.   (d) \(\ln(t^2)=2\ln t\) (since \(t>0\)), then \(u=\ln t\).   (e) \(\sin(2t)=2\sin t\cos t\) to cancel \(\cos t\).

PART 2 — Strategy & Tips (roadmap only)

Plan • Pitfalls • Quick wins

Minimal roadmap (no algebra yet)

  1. (a) Split the polynomial; integrate each term with the power rule; evaluate at \(0\) and \(\sqrt5\).
  2. (b) Let \(u=1+x^2\) so \(du=2x\,dx\); update bounds; integrate \(u^{-1/2}\).
  3. (c) Check parity: \(\sin(\theta)\) is odd ⇒ \(\sin^7(\theta)\) is odd ⇒ symmetric integral is zero.
  4. (d) Use \(t>0\) to rewrite \(\ln(t^2)=2\ln t\); then set \(u=\ln t\) so \(du=dt/t\).
  5. (e) Use \(\sin(2t)=2\sin t\cos t\) to simplify; then integrate \(\sin t\) over \([0,\pi/4]\).

Common mistakes

  • (b) Forgetting the factor \(1/2\) from \(du=2x\,dx\).
  • (c) Misclassifying odd vs even functions.
  • (d) Using \(\ln(t^2)=2\ln t\) without checking \(t>0\).
  • (e) Not simplifying \(\sin(2t)\) before integrating.

Quick tips

  • If you see x dx next to 1+x², try \(u=1+x^2\).
  • If bounds are symmetric, test odd/even immediately.
  • Always simplify trig first; it’s often the whole trick.

PART 3 — Full Solution

Intuition • Derivation • Box • Checks

Physical intuition (why definite integrals behave this way)

A definite integral is “total accumulation” over an interval (area under a curve). If the integrand stays positive, the result must be positive. If the integrand is odd over symmetric limits, positive and negative contributions cancel, giving zero. Substitution is like choosing a better coordinate so the accumulation becomes simple.

(a) Polynomial integral — use linearity + power rule Method: power rule

Integral: \(\displaystyle \int_{0}^{\sqrt5}(1+2x+3x^2+4x^3+5x^4)\,dx\)

∫(1)dx = x ∫(2x)dx = x² ∫(3x²)dx = x³ ∫(4x³)dx = x⁴ ∫(5x⁴)dx = x⁵

So an antiderivative is: F(x) = x + x² + x³ + x⁴ + x⁵

Evaluate with FTC: F(√5) − F(0)

(√5)² = 5 (√5)³ = 5√5 (√5)⁴ = 25 (√5)⁵ = 25√5
F(√5) = √5 + 5 + 5√5 + 25 + 25√5 = (5+25) + (1+5+25)√5 = 30 + 31√5
Final boxed answer
(a) 30 + 31√5
Units

Pure math: consistent. In applications, coefficients would carry units so the sum matches “integrand units,” then multiply by \(dx\).

Limits

Upper limit \(\sqrt5\approx 2.236\): polynomial grows; result in the “tens” is plausible.

Interpretation

Integrand is positive on \([0,\sqrt5]\) ⇒ accumulated area must be positive; answer is positive.

(b) Rational + square root — use \(u\)-substitution Method: u-sub

Integral: \(\displaystyle \int_{0}^{10}\frac{x}{\sqrt{1+x^2}}\,dx\)

Choose \(u=1+x^2\). Then \(du=2x\,dx\) so \(x\,dx=\frac12 du\).

u = 1 + x² du = 2x dx ⇒ x dx = (1/2) du

Update bounds: \(x=0\Rightarrow u=1\), \(x=10\Rightarrow u=101\).

∫[0,10] x/√(1+x²) dx = (1/2) ∫[1,101] u^(-1/2) du
∫ u^(-1/2) du = 2u^(1/2) ⇒ (1/2)[2u^(1/2)] = √u
√u |[1→101] = √101 − 1
Final boxed answer
(b) √101 − 1
Units

If \(x\) is dimensionless, result is dimensionless. In physics, \(1+x^2\) would imply a scaled variable.

Limits

For large \(x\), \(x/√(1+x^2)\approx 1\), so integral \(\approx 10\). Here \(\sqrt{101}-1\approx 9.05\): reasonable.

Interpretation

Integrand is nonnegative on \([0,10]\) ⇒ accumulation must be nonnegative; answer is positive.

(c) Symmetry integral — odd function cancels Method: symmetry

Integral: \(\displaystyle \int_{-\pi/2}^{\pi/2}\sin^{7}(\theta)\,d\theta\)

Since \(\sin(-\theta)=-\sin(\theta)\), we have:

sin(-θ) = -sin(θ) ⇒ sin⁷(-θ) = (-sin(θ))⁷ = -sin⁷(θ)

So \(\sin^7(\theta)\) is odd. For any odd function \(f\), \(\int_{-a}^{a} f(\theta)\,d\theta=0\) because areas cancel.

Final boxed answer
(c) 0
Units

Angles are dimensionless in trig; the integral is dimensionless.

Limits

Symmetric bounds \([-\pi/2,\pi/2]\) is exactly when odd-cancellation applies.

Interpretation

Positive contribution for \(\theta>0\) equals negative for \(\theta<0\) ⇒ net accumulation zero.

(d) Log integral — simplify log, then \(u=\ln t\) Method: log + u-sub

Integral: \(\displaystyle \int_{1}^{2}\frac{\ln(t^2)}{t}\,dt\)

On \([1,2]\), \(t>0\). Therefore \(\ln(t^2)=2\ln t\).

∫[1,2] ln(t²)/t dt = ∫[1,2] 2 ln(t)/t dt

Now set \(u=\ln t\). Then \(du=\frac{dt}{t}\).

u = ln t du = (1/t) dt

Update bounds: \(t=1\Rightarrow u=0\), \(t=2\Rightarrow u=\ln 2\).

∫[1,2] 2 ln(t)/t dt = 2 ∫[0, ln2] u du
2 ∫ u du = 2 * (u²/2) = u²
u² |[0→ln2] = (ln 2)²
Final boxed answer
(d) (ln 2)²
Units

Logarithms require dimensionless arguments; here it’s a pure math integral. Result is dimensionless.

Limits

On \([1,2]\), \(\ln(t^2)\ge 0\) and \(1/t>0\) ⇒ integral must be positive; \((\ln2)^2>0\).

Interpretation

This is the accumulated “log-weighted density” \( \ln(t^2)\) scaled by \(1/t\); substitution makes it a simple area in \(u\).

(e) Trig simplification — identity cancels the denominator Method: trig identity

Integral: \(\displaystyle \int_{0}^{\pi/4}\frac{4\sin(2t)}{\cos t}\,dt\)

Use \(\sin(2t)=2\sin t\cos t\). On \([0,\pi/4]\), \(\cos t>0\), so cancellation is safe.

4 sin(2t)/cos(t) = 4*(2 sin t cos t)/cos t = 8 sin t
∫[0,π/4] 8 sin t dt = 8 ∫[0,π/4] sin t dt = 8 [-cos t]_[0→π/4]
= 8( -cos(π/4) + cos(0) ) = 8( 1 - √2/2 ) = 8 - 4√2
Final boxed answer
(e) 8 − 4√2
Units

Angles are dimensionless; the integral is dimensionless.

Limits

Integrand is positive on \([0,\pi/4]\) ⇒ result positive. Numerically \(8-4\sqrt2\approx 2.34>0\).

Interpretation

Identity converts a “complicated” ratio into a simple sinusoid accumulation.
