Problem 2 — Right-Endpoint Riemann Sum & a Limit via an Integral

We approximate ∫₀ᵃ 1/√(1+x²) dx using a right-endpoint partition, then recognize a “mysterious” limit as that same idea in disguise.

Given antiderivative: ln(x + √(x²+1))
Rule: Right endpoints
Target limit: Σ 1/√(9n²+16k²)

Quick Results

Part (a): right-endpoint sum for the integral.

Part (b): rewrite the limit as an integral and evaluate.

(1/4) ln 3 ≈ 0.27465

PART 1 — Problem Analysis (no solving yet)

analysis

We are given the function f(x)=1/√(1+x²) and the antiderivative formula ∫ dx/√(1+x²)=ln(x+√(x²+1)) (for x>0).

Interpretation: The definite integral is the “accumulated area” under a positive curve. A Riemann sum replaces that area by thin rectangles (height from the curve, width from the partition).

What we must do:

  • (a) Partition [0,a] into n equal pieces, use right endpoints, and write the Riemann sum for ∫₀ᵃ f(x) dx.
  • (b) Choose an appropriate a so the limit looks like that Riemann sum as n→∞, then evaluate the integral using the given antiderivative.

Key tools / why they apply:

  • Riemann sum definition: ∫₀ᵃ f(x)dx = limₙ→∞ Σ f(x_k*) Δx.
  • Uniform partition: equal subintervals → constant Δx.
  • Scaling in sums: factor out n to reveal (1/n) and (k/n).

Approaches (choose best):

  • Direct construction of the right-endpoint sum (best for part a).
  • Use the antiderivative to evaluate the associated integral (needed for part b).
  • Hyperbolic viewpoint (optional): same integral equals arsinh(x); not required.
Common pitfall: Using k=0…n-1 gives left endpoints. Right endpoints use k=1…n.

PART 2 — Strategy & Tips (roadmap only)

roadmap
Minimal roadmap (no algebra yet) ›
  1. Compute the width of each subinterval. Tool: Δx = a/n.
  2. Write right endpoints. Tool: x_k = kΔx, for k=1…n.
  3. Evaluate f at right endpoints. Tool: substitute x_k into f(x)=1/√(1+x²).
  4. Form the Riemann sum. Tool: R_n = Σ f(x_k) Δx.
  5. For the limit, factor n² inside the square root to expose 1/n and k/n.
  6. Identify the integral as ∫₀¹ g(x) dx for some g.
  7. Evaluate the integral by a substitution to match ∫ du/√(1+u²), then apply the log formula.
Tip: In part (b), you want the sum to look like Σ g(k/n) · (1/n) because that is the signature of ∫₀¹ g(x) dx.

PART 3 — Full Solution

solution
Physical intuition: Think “accumulation.” The integral is a continuous sum of tiny contributions \(f(x)\,dx\). The Riemann sum approximates it with rectangles: height sampled on the curve, width equals the slice size.

(a) Right-endpoint Riemann sum for ∫₀ᵃ 1/√(1+x²) dx

Partition [0,a] into n equal pieces.

Step 1: Δx = (a - 0)/n = a/n Step 2: Right endpoints: x_k = 0 + kΔx = k(a/n), for k = 1,2,...,n Step 3: f(x_k) = 1 / √(1 + x_k²) = 1 / √(1 + (ka/n)²) Step 4: R_n = Σ_{k=1}^{n} f(x_k)Δx = Σ_{k=1}^{n} [ 1/√(1 + (ka/n)²) ] · (a/n)
Final (a)
R_n = (a/n) · Σ_{k=1}^{n} 1 / √( 1 + (ka/n)² )

(b) Evaluate limₙ→∞ Σ_{k=1}^n 1/√(9n²+16k²)

The goal is to rewrite the sum in the signature form Σ g(k/n)·(1/n).

Step 1: Factor n² inside the square root: √(9n² + 16k²) = √(n²[9 + 16(k/n)²]) = n √(9 + 16(k/n)²) So each term becomes: 1/√(9n² + 16k²) = 1/[ n √(9 + 16(k/n)²) ] = (1/n) · 1/√(9 + 16(k/n)²) Therefore: Σ_{k=1}^{n} 1/√(9n²+16k²) = Σ_{k=1}^{n} (1/n) · 1/√(9 + 16(k/n)²)

This is the right-endpoint Riemann sum for ∫₀¹ 1/√(9+16x²) dx.

lim_{n→∞} Σ_{k=1}^{n} (1/n) · 1/√(9 + 16(k/n)²) = ∫_{0}^{1} 1/√(9 + 16x²) dx

Now evaluate the integral using the provided antiderivative \(∫ du/√(1+u²)\).

Let u = (4x)/3 ⇒ x = (3u)/4 and dx = (3/4) du. Compute the denominator: √(9 + 16x²) = √(9 + 16( (3u)/4 )² ) = √(9 + 9u²) = 3 √(1 + u²) So: ∫₀¹ 1/√(9+16x²) dx = ∫_{u=0}^{u=4/3} [ 1 / (3√(1+u²)) ] · (3/4) du = (1/4) ∫₀^{4/3} 1/√(1+u²) du Use the given antiderivative: ∫ 1/√(1+u²) du = ln( u + √(1+u²) ) Thus: (1/4)[ ln(u + √(1+u²)) ]₀^{4/3} = (1/4)( ln( 4/3 + √(1+16/9) ) - ln(1) ) √(1+16/9) = √(25/9) = 5/3 ⇒ 4/3 + 5/3 = 3 So the value is: = (1/4) ln(3)
Final (b)
lim_{n→∞} Σ_{k=1}^{n} 1/√(9n² + 16k²) = (1/4) ln 3
Sanity checks (units • limits • interpretation) ›
Units: The expressions are dimensionless (pure numbers). The result (1/4)ln 3 is also dimensionless — consistent.
Reasonableness: In (b), each term is about ~ 1/(n·constant). With n terms, the sum tends to a finite constant — as found.
Bounding check: For x∈[0,1], 1/√(9+16x²) is between 1/5 and 1/3, so the integral is between 0.2 and 0.333…. Our answer (1/4)ln 3 ≈ 0.27465 fits nicely.
Physical interpretation: The limit is a continuous “accumulated contribution” obtained by refining a partition. It is literally the area under a specific curve on [0,1].
