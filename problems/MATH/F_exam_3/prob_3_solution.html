PART 1 — Problem Analysis (no solving yet)

Understand → choose method

Restate the problem

Solve the initial-value problem:

\[ \begin{cases} y'(x)=e^{x+2y},\\ y(0)=0. \end{cases} \]

Find an explicit \(y(x)\) and state where it is valid (real-valued).

Given / Unknowns / Goal

  • Given: \(y'(x)=e^{x+2y}\)
  • Given: \(y(0)=0\)
  • Unknown: \(y(x)\)
  • Goal: explicit solution + validity interval

Why the main principle applies

The equation is separable because the right-hand side factorizes: \(e^{x+2y}=e^x\,e^{2y}\). That means we can rearrange it into “all \(y\)” on one side and “all \(x\)” on the other, then integrate.

Candidate approaches

  • Separation of variables: fastest because it factorizes immediately.
  • Substitution: let \(u=e^{-2y}\), turns into a simple ODE for \(u\).
  • Implicit-first approach: separate and integrate, then solve for \(y\).

Chosen approach

Separation of variables is the cleanest: fewest steps, minimal algebra, and the initial condition fixes the constant without ambiguity.

PART 2 — Strategy & Tips (roadmap only)

Plan → avoid mistakes

Factor the exponential

Goal: make separability obvious.
Tool: \(e^{a+b}=e^a e^b\).

Separate variables

Goal: move \(y\)-terms to the left, \(x\)-terms to the right.
Tool: algebra with \(dy/dx\).

Integrate both sides

Goal: compute \(\int e^{-2y}dy\) and \(\int e^x dx\).
Tool: exponential integrals, chain rule awareness.

Apply the initial condition

Goal: determine the integration constant.
Tool: substitute \(x=0, y=0\).

Solve for \(y(x)\)

Goal: isolate \(y\) using logarithms.
Tool: \(\ln\) and exponent rules.

State the real domain

Goal: ensure the log argument is positive.
Tool: inequality check.

Verify + interpret

Goal: differentiate back and confirm the ODE; interpret blow-up.
Tool: implicit differentiation + limits.

Common mistakes: (i) missing the factor \(-\tfrac12\) in \(\int e^{-2y}dy\), (ii) forgetting \(e^{x+2y}=e^x e^{2y}\), (iii) ignoring the domain where \(\ln(\cdot)\) is real.

PART 3 — Full Solution

Intuition → derivation → final

Physical intuition: The rate \(y'(x)=e^{x+2y}\) is always positive and grows extremely fast as \(x\) or \(y\) increases. This “exponential feedback” often causes runaway growth and can blow up at a finite value of \(x\).

Derivation (step-by-step)

Start by factorizing the exponential:

\[ y' = e^{x+2y} = e^x e^{2y}. \]

Write \(y'=\dfrac{dy}{dx}\) and separate variables:

\[ \frac{dy}{dx} = e^x e^{2y} \quad\Longrightarrow\quad e^{-2y}\,dy = e^x\,dx. \]

Integrate both sides:

\[ \int e^{-2y}\,dy = \int e^x\,dx. \]

Compute each integral carefully (chain rule on the left):

\[ \int e^{-2y}\,dy = -\frac12 e^{-2y}, \qquad \int e^x\,dx = e^x + C. \] \[ -\frac12 e^{-2y} = e^x + C. \]

Apply the initial condition \(y(0)=0\):

\[ -\frac12 e^{-2\cdot 0} = e^0 + C \quad\Longrightarrow\quad -\frac12 = 1 + C \quad\Longrightarrow\quad C = -\frac32. \]

Substitute back:

\[ -\frac12 e^{-2y} = e^x - \frac32. \]

Solve explicitly for \(y(x)\):

\[ e^{-2y} = -2e^x + 3 = 3 - 2e^x. \] \[ -2y = \ln(3 - 2e^x) \quad\Longrightarrow\quad y(x) = -\frac12 \ln(3 - 2e^x). \]

Final Answer

\[ \boxed{\,y(x)= -\frac12 \ln\!\bigl(3-2e^x\bigr)\,} \]

Real-valued solution requires \(3-2e^x>0\), i.e. \[ \boxed{\,x<\ln\!\left(\frac32\right)\,}. \] 

\boxed{\,y(x)= -\frac12 \ln\!\bigl(3-2e^x\bigr)\,},\qquad x<\ln\!\left(\frac32\right).

Sanity Checks (units • limits • verification • interpretation)

Trust the result
✓

Initial condition: \(y(0)=-\tfrac12\ln(3-2\cdot 1)=-\tfrac12\ln(1)=0\).

✓

ODE verification (outline): From \(e^{-2y}=3-2e^x\), differentiate: \(-2y'e^{-2y}=-2e^x \Rightarrow y'=e^x e^{2y}=e^{x+2y}\).

!

Units: Exponents/logs require dimensionless arguments. In this exam-style ODE, \(x\) and \(y\) are treated as dimensionless, so \(\ln(3-2e^x)\) is consistent.

⟂

Finite-time blow-up: As \(x\to \ln(3/2)^-\), \(3-2e^x\to 0^+\), hence \(\ln(\cdot)\to -\infty\) and \(y(x)\to +\infty\). This matches the “runaway” intuition from exponential feedback.

Physical interpretation (optional): If \(y\) were a population/energy-like quantity with growth rate exponentially amplified by both position/time \(x\) and the state \(y\), then the model predicts rapid acceleration and a divergence at a finite \(x\). In real physics, such blow-up usually signals missing damping/saturation mechanisms.

Common Pitfalls
