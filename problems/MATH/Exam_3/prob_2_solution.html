Problem 2 — Accumulation Function with a Moving Upper Limit

We study an integral-defined function and extract its derivative, then the closed form, and finally justify the classic antiderivative of \( \frac{1}{1+x^2} \) — all from first principles.

Given
\[ F(x)=\int_{0}^{\tan x}\frac{1}{1+t^{2}}\,dt,\qquad 0\le x<\frac{\pi}{2}. \]
Tasks
(a) Find \(F'(x)\) using FTC (not a table).
(b) Use (a) to get \(F(x)\).
(c) Use (b) to justify an antiderivative of \(\frac{1}{1+x^2}\).

Core identity
\[ 1+\tan^2x=\sec^2x \]

This is the cancellation that makes \(F'(x)\) become a constant.

PART 1

Problem Analysis (no solving yet)

Rewrite the problem

Define \[ F(x)=\int_{0}^{\tan x}\frac{1}{1+t^{2}}\,dt \quad (0\le x<\tfrac{\pi}{2}). \] (a) Compute \(F'(x)\) using FTC + chain rule. (b) Use (a) to find \(F(x)\). (c) Use (b) to justify an antiderivative of \( \frac{1}{1+x^2} \).

Given / Unknowns / Must find

  • Given: integrand \(f(t)=\frac{1}{1+t^2}\), upper limit \(g(x)=\tan x\), domain \(0\le x<\frac{\pi}{2}\).
  • Unknowns: \(F'(x)\), formula for \(F(x)\), antiderivative of \(\frac{1}{1+x^2}\).
  • Must show: FTC use + simplification (no antiderivative table for part (a)).

Relevant principle (why it applies)

This is an accumulation function with a moving endpoint. The Fundamental Theorem of Calculus (Part 1) tells us the derivative equals “integrand at the endpoint” times “endpoint speed”:

\[ \frac{d}{dx}\left(\int_{a}^{g(x)} f(t)\,dt\right)=f(g(x))\cdot g'(x). \]

We also rely on the trig identity \(1+\tan^2x=\sec^2x\) to simplify.

Approach A (best)

FTC + chain rule (exactly what is requested, minimal steps).

Approach B

Evaluate integral using \(\arctan t\) then differentiate (works, but not allowed for (a)).

Approach C

Substitute \(t=\tan u\) to rewrite integral (insightful but longer).

PART 2

Strategy & Tips (roadmap only)

  1. Match the FTC pattern
    Identify \(f(t)\) and \(g(x)\) so you can use FTC Part 1.
  2. Differentiate with FTC + chain rule
    Write \(F'(x)=f(g(x))\cdot g'(x)\).
  3. Compute \(g'(x)\)
    Differentiate \(g(x)=\tan x\Rightarrow g'(x)=\sec^2x\).
  4. Simplify using a trig identity
    Use \(1+\tan^2x=\sec^2x\) to cancel factors.
  5. Recover \(F(x)\)
    If \(F'(x)=1\), then \(F(x)=x+C\).
  6. Fix the constant
    Evaluate \(F(0)\) from the definition to find \(C\).
  7. Extract the antiderivative
    Define \(G(u)=\int_0^u \frac{1}{1+t^2}dt\). Then \(G'(u)=\frac{1}{1+u^2}\) and relate \(G\) to \(F\).

Common mistakes

  • Forgetting the chain rule factor \(g'(x)\).
  • Substituting \(t=x\) inside the integrand (wrong variable) before applying FTC.
  • Missing \(1+\tan^2x=\sec^2x\) — without it, simplification stalls.
  • Ignoring the constant \(C\) in \(F(x)=x+C\); always use \(F(0)\).
PART 3

Full Solution

Physical intuition (rate of accumulation)

Think of \(F(x)\) as accumulated area under \( \frac{1}{1+t^2} \) up to the moving endpoint \(t=\tan x\). The instantaneous rate of change is:

\[ \text{(height at endpoint)}\times\text{(speed of endpoint)} = \frac{1}{1+\tan^2x}\cdot \frac{d}{dx}(\tan x). \]

The identity \(1+\tan^2x=\sec^2x\) makes these factors cancel, so the area grows at a constant rate.

(a) Compute \(F'(x)\)

Let \(f(t)=\frac{1}{1+t^2}\) and \(g(x)=\tan x\). By FTC Part 1:

\[ F'(x)=f(g(x))\,g'(x) = \frac{1}{1+(\tan x)^2}\cdot \frac{d}{dx}(\tan x). \]

Differentiate \(\tan x\): \(\frac{d}{dx}(\tan x)=\sec^2x\). So

\[ F'(x)=\frac{1}{1+\tan^2x}\cdot \sec^2x. \]

Use \(1+\tan^2x=\sec^2x\):

\[ F'(x)=\frac{1}{\sec^2x}\cdot \sec^2x=1 \quad\Rightarrow\quad \boxed{F'(x)=1}. \]

(b) Find \(F(x)\)

Since \(F'(x)=1\), \(F\) must be linear:

\[ F(x)=x+C. \]

Find \(C\) using \(F(0)\). Since \(\tan 0=0\):

\[ F(0)=\int_0^0 \frac{1}{1+t^2}\,dt=0. \]

So \(0=0+C\Rightarrow C=0\). Therefore:

\[ \boxed{F(x)=x \quad \text{for } 0\le x<\frac{\pi}{2}.} \]

(c) Antiderivative of \(\frac{1}{1+x^2}\) from (b)

Define the standard accumulation function:

\[ G(u)=\int_0^{u}\frac{1}{1+t^2}\,dt. \]

By FTC Part 1:

\[ G'(u)=\frac{1}{1+u^2}. \]

Now observe that your original function is \(F(x)=G(\tan x)\). From (b), \(F(x)=x\), hence:

\[ G(\tan x)=x \quad (0\le x<\tfrac{\pi}{2}). \]

This means \(G\) is the inverse function of \(\tan\) on this interval, i.e. \(G(u)=\arctan(u)\). Therefore an antiderivative is:

\[ \boxed{\int \frac{1}{1+x^2}\,dx=\arctan(x)+C.} \]
SUMMARY

Final Results + Sanity Checks

Derivative
\(\boxed{F'(x)=1}\)
Closed form
\(\boxed{F(x)=x}\)
Antiderivative
\(\boxed{\int \frac{1}{1+x^2}\,dx=\arctan(x)+C}\)

Sanity checks

  • Units: dimensionless math objects; consistent.
  • Limit \(x\to 0\): \(\tan x\to 0\Rightarrow F(x)\to 0\) matches \(F(x)=x\).
  • Limit \(x\to \frac{\pi}{2}^-\): \(\tan x\to \infty\Rightarrow F(x)\to \frac{\pi}{2}\), consistent with total area under \(\frac{1}{1+t^2}\) from \(0\) to \(\infty\).
  • Interpretation: endpoint speed \(\sec^2x\) cancels endpoint height \(1/(1+\tan^2x)\), giving constant accumulation rate.
