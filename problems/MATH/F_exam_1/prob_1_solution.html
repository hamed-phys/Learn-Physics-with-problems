PART 3 â€” Full Solution

Derivation

Physical intuition

The function \(y=x+\frac{1}{x}\) has two competing behaviors: the \(x\) term increases with slope \(1\), while \(\frac{1}{x}\) decreases for \(x>0\). For large \(x\), \(\frac{1}{x}\) changes slowly, so the slope should be close to \(1\). For very small \(x>0\), \(\frac{1}{x}\) changes rapidly, so the slope should become very negative.

Step-by-step derivation from first principles

\[ \text{Given } y(x)=x+\frac{1}{x}, \quad x>0. \] \[ y'(x)=\lim_{h\to 0}\frac{y(x+h)-y(x)}{h}. \] \[ y(x+h)=(x+h)+\frac{1}{x+h}. \] \[ y(x+h)-y(x)=\left[(x+h)+\frac{1}{x+h}\right]-\left[x+\frac{1}{x}\right] =h+\left(\frac{1}{x+h}-\frac{1}{x}\right). \] \[ \frac{1}{x+h}-\frac{1}{x}=\frac{x-(x+h)}{x(x+h)}=\frac{-h}{x(x+h)}. \] \[ \frac{y(x+h)-y(x)}{h} =\frac{h-\frac{h}{x(x+h)}}{h} =\frac{h\left(1-\frac{1}{x(x+h)}\right)}{h} =1-\frac{1}{x(x+h)}. \] \[ \Rightarrow\quad y'(x)=\lim_{h\to 0}\left(1-\frac{1}{x(x+h)}\right) =1-\frac{1}{x^2}. \]

Final result

\[ \boxed{\frac{dy}{dx}=1-\frac{1}{x^{2}},\qquad x>0.} \]

Sanity checks

Units check
Mathematically, \(\frac{dy}{dx}\) is a slope. In a physical model, adding \(x\) and \(\frac{1}{x}\) only makes unit-sense if \(x\) is dimensionless (or scaled). Under that assumption, the derivative \(1-\frac{1}{x^2}\) is dimensionless and consistent.
Limiting cases behavior
  • As \(x\to \infty\): \(\;y'(x)=1-\frac{1}{x^2}\to 1\). The slope approaches the slope of \(y\approx x\).
  • As \(x\to 0^+\): \(\;y'(x)=1-\frac{1}{x^2}\to -\infty\). The \(\frac{1}{x}\) term dominates and decreases extremely fast.
Physical interpretation meaning
If \(x\) were time and \(y\) were position, then \(y'(x)\) is instantaneous velocity. This result says velocity is nearly \(1\) for large \(x\), but becomes strongly negative when \(x\) is very small and positiveâ€”matching the intuition that \(1/x\) changes rapidly near \(0^+\).
