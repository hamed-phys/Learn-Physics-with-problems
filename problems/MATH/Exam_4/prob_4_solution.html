Problem 4 — Arc Length (Parametric Curve + DE-Given Slope)

Two arc-length computations: (a) a curve given parametrically on \(0\le t\le \pi\), and (b) a curve \(y=f(x)\) known only through its derivative \(y'(x)\) on \(2\le x\le 3\), where you are explicitly told not to solve the differential equation.

Problem Statement

(a) Given \(x(t)=\cos t + t\sin t\), \(y(t)=\sin t - t\cos t\), compute the arc length for \(0\le t\le \pi\).

(b) Given \(y'=\sqrt{x^2-1}\), \(y(2)=0\), compute the arc length of \(y=f(x)\) on \(2\le x\le 3\) without solving for \(y(x)\).

Physics Lens

Arc length is literally distance traveled. If a particle’s position is \(\mathbf r(t)\), then distance is \(\int \|\mathbf r'(t)\|\,dt\). If a path is graphed as \(y(x)\), then \(ds=\sqrt{dx^2+dy^2}=\sqrt{1+(y')^2}\,dx\).

∥v∥

Why these formulas apply: The infinitesimal distance in the plane is \(ds=\sqrt{dx^2+dy^2}\). Parametrizing by time \(t\) turns this into speed: \(ds=\|\mathbf r'(t)\|\,dt\). Parametrizing by \(x\) turns it into slope: \(ds=\sqrt{1+(dy/dx)^2}\,dx\).

3) Candidate approaches

  • (a) Use parametric arc length formula directly (cleanest).
  • (a) Vector-speed method \(\|\mathbf r'(t)\|\) (same math, clearer physics).
  • (b) Plug \(y'\) into \(\sqrt{1+(y')^2}\) (obeys “don’t solve DE”).

4) Best choice (and why)

  • (a) Differentiate \(x(t),y(t)\), simplify \(\sqrt{(x')^2+(y')^2}\), integrate over \(t\).
  • (b) Use arc length in terms of \(y'\) only; the initial condition isn’t needed for length.

Tips Quick wins

  • Think “distance = integral of speed” for parametric curves.
  • Always simplify \(\sqrt{(x')^2+(y')^2}\) before integrating.
  • Remember \(\sqrt{u^2}=|u|\). Use the interval to drop \(|\cdot|\) safely.

Avoid Common mistakes

  • Forgetting product rule on \(t\sin t\) or \(t\cos t\).
  • Dropping the absolute value \(\sqrt{t^2}\to t\) without checking the interval.
  • Solving the DE in (b) even though it’s unnecessary (and forbidden).

PART 3 — Full Solution

Rigorous + Beginner-Friendly
ds

Physical intuition: In the plane, a tiny move has length \(ds=\sqrt{dx^2+dy^2}\). If the motion is described by time \(t\), then \(dx=x'(t)\,dt\), \(dy=y'(t)\,dt\), so the speed is \(\|\mathbf r'(t)\|\). If the curve is described as \(y(x)\), then \(dy=y'(x)\,dx\), giving \(ds=\sqrt{1+(y')^2}\,dx\).

(a) Parametric curve arc length

We use the parametric arc length formula:

\[ L=\int_{0}^{\pi}\sqrt{\left(\frac{dx}{dt}\right)^2+\left(\frac{dy}{dt}\right)^2}\,dt. \]

Differentiate \(x(t)=\cos t+t\sin t\):

\[ \frac{dx}{dt}=\frac{d}{dt}(\cos t)+\frac{d}{dt}(t\sin t) =(-\sin t)+(\sin t+t\cos t)=t\cos t. \]

Differentiate \(y(t)=\sin t-t\cos t\):

\[ \frac{dy}{dt}=\frac{d}{dt}(\sin t)-\frac{d}{dt}(t\cos t) =\cos t-(\cos t-t\sin t)=t\sin t. \]

Now compute the speed:

\[ \sqrt{(x')^2+(y')^2} =\sqrt{(t\cos t)^2+(t\sin t)^2} =\sqrt{t^2(\cos^2 t+\sin^2 t)} =\sqrt{t^2} =|t|. \]

Since \(0\le t\le \pi\), we have \(|t|=t\). Therefore:

\[ L=\int_{0}^{\pi} t\,dt =\left[\frac{t^2}{2}\right]_{0}^{\pi} =\frac{\pi^2}{2}. \]

Final Answer (a)

Arc Length
\[ \boxed{L_a=\frac{\pi^2}{2}} \]
›Sanity checks (a)
  • Units: If \(x,y\) have length units and \(t\) is dimensionless, the integral returns length units.
  • Limiting case: For \(0\le t\le T\), the same algebra gives \(L=\tfrac{T^2}{2}\), which matches “distance = area under speed” when \(v(t)=t\).
  • Interpretation: The speed is \(v(t)=t\): starts at 0 and grows linearly, so distance is triangular area \(\tfrac12(\pi)(\pi)\).
(b) Arc length using only the derivative \(y'(x)\)

For a curve written as \(y=f(x)\), the arc length on \([2,3]\) is:

\[ L=\int_{2}^{3}\sqrt{1+\left(\frac{dy}{dx}\right)^2}\,dx. \]

We are given \(\dfrac{dy}{dx}=\sqrt{x^2-1}\). Substitute it directly:

\[ 1+\left(\frac{dy}{dx}\right)^2 =1+\left(\sqrt{x^2-1}\right)^2 =1+(x^2-1) =x^2. \]

So the integrand becomes:

\[ \sqrt{1+(y')^2}=\sqrt{x^2}=|x|. \]

On \([2,3]\), \(x>0\), hence \(|x|=x\). Therefore:

\[ L=\int_{2}^{3}x\,dx =\left[\frac{x^2}{2}\right]_{2}^{3} =\frac{9}{2}-\frac{4}{2} =\frac{5}{2}. \]

Final Answer (b)

Arc Length
\[ \boxed{L_b=\frac{5}{2}} \]
›Sanity checks (b)
  • Units: \(\sqrt{1+(y')^2}\) is dimensionless (if \(x,y\) share units), so integrating over \(dx\) gives length.
  • Absolute value: \(\sqrt{x^2}=|x|\) is crucial; it simplifies to \(x\) only because \(x\in[2,3]\).
  • Interpretation: Since \(y'=\sqrt{x^2-1}\) grows with \(x\), the curve steepens as \(x\) increases, so each unit of \(x\) corresponds to more distance along the curve.
›Why the initial condition \(y(2)=0\) is not needed for length
The arc length formula depends only on the slope \(y'(x)\) through \(\sqrt{1+(y')^2}\). A vertical shift in \(y\) changes the curve’s position but not its local stretching with respect to \(x\), so the total length on \([2,3]\) is determined entirely by \(y'(x)\).
