Restate: Analyze the rational function \(y=\frac{x^3}{x^2-1}\) and sketch it on \((-3,3)\).
Given: \(y=\dfrac{x^3}{x^2-1}\), interval \((-3,3)\).
Unknowns: discontinuities + type, asymptotes, end behavior, local maxima/minima.
Must find: exact features + enough key points/behavior to sketch correctly.
Tools (why):
  • Domain / poles: discontinuities occur where denominator is zero.
  • Asymptotics: compare leading degrees → slant/horizontal behavior.
  • Derivative: extrema where \(y'(x)=0\) (and function is defined).
Approaches:
  • Full calculus workflow (domain → asymptotes → derivative → sign chart → sketch).
  • “Transfer-function” intuition (poles + high-\(|x|\) trend).
  • Pure sampling (fast but risky: can miss extrema/asymptotes).
Best choice: calculus workflow — gives exact discontinuities, asymptotes, and extrema.

PART 2 — Strategy & Tips (roadmap only)

  1. Domain: solve \(x^2-1\neq 0\). (Tool: algebra)
  2. Classify discontinuities: check cancellation vs blow-up. (Tool: factor check + limits)
  3. Vertical asymptotes: from denominator roots (no cancellation). (Tool: limit behavior)
  4. Slant asymptote: long division / rewrite. (Tool: polynomial division)
  5. Derivative: compute \(y'(x)\) and solve \(y'=0\) in domain. (Tool: derivative)
  6. Classify extrema: sign chart or 2nd derivative. (Tool: sign analysis)
  7. Plot key points: intercepts + extrema + near-asymptote behavior. (Tool: evaluation)
  8. Sketch: draw asymptotes first, then curves with monotonic intervals. (Tool: structure)
Common mistakes:
  • Forgetting \(x=\pm 1\) are excluded from the domain.
  • Assuming horizontal asymptote (here it’s slant).
  • Messing up one-sided limits near \(x=\pm 1\).

PART 3 — Full Solution

Intuition: denominator zeros behave like “poles” → blow-up near \(x=\pm1\). Degree difference 1 → follows a line \(y\approx x\) for large \(|x|\).
1) Domain + discontinuities
\(x^2-1\neq 0 \Rightarrow x\neq \pm1.\) No cancellation occurs \(\Rightarrow\) infinite discontinuities.
2) Asymptotes
  • Vertical: \(x=1\), \(x=-1\).
  • Slant: \( \displaystyle \frac{x^3}{x^2-1}=x+\frac{x}{x^2-1}\Rightarrow y=x.\)
3) One-sided behavior near vertical asymptotes
  • \(\lim_{x\to 1^-} y=-\infty,\quad \lim_{x\to 1^+} y=+\infty\)
  • \(\lim_{x\to (-1)^-} y=-\infty,\quad \lim_{x\to (-1)^+} y=+\infty\)
4) Symmetry + intercept
\(y(-x)=-y(x)\Rightarrow\) odd symmetry. Also \(y(0)=0\Rightarrow(0,0)\) is an intercept.
5) Extrema
Using \(y=x+\dfrac{x}{x^2-1}\), \[ y' = 1 - \frac{x^2+1}{(x^2-1)^2} = \frac{x^2(x^2-3)}{(x^2-1)^2}. \] Critical points: \(x=0,\ x=\pm\sqrt{3}\) (valid since \(\pm\sqrt{3}\neq \pm1\)). Sign analysis shows:
  • Local max at \(x=-\sqrt{3}\): \(\displaystyle y=-\frac{3\sqrt{3}}{2}\approx -2.60\)
  • Local min at \(x=+\sqrt{3}\): \(\displaystyle y=\frac{3\sqrt{3}}{2}\approx 2.60\)
  • \((0,0)\) is stationary but not an extremum (no sign flip).
Final boxed features
  • Domain holes / discontinuities: \(x=\pm1\) (infinite discontinuities)
  • Vertical asymptotes: \(x=-1,\ x=1\)
  • Slant asymptote: \(y=x\)
  • Local max: \(\left(-\sqrt3,\ -\frac{3\sqrt3}{2}\right)\approx(-1.73,-2.60)\)
  • Local min: \(\left(\sqrt3,\ \frac{3\sqrt3}{2}\right)\approx(1.73,2.60)\)
  • Stationary point: \((0,0)\)
