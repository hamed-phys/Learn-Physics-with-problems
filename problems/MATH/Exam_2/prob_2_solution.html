Problem 2 — Closest point on \(y=\frac12x^2\) to \(P(4,1)\)

Find the point \(Q\) on the parabola minimizing Euclidean distance to \(P\).

file: prob2.html

PART 1 — Problem Analysis (no solving yet)

Restate: Minimize distance from \(P(4,1)\) to a point \(Q(x,y)\) constrained by \(y=\frac12x^2\).
Given: curve \(y=\frac12x^2\), point \(P(4,1)\).
Unknowns: closest point \(Q=(x,y)\) on curve.
Goal: find coordinates of \(Q\).
Key ideas:
  • Minimize \(D^2=(x-4)^2+(y-1)^2\) instead of \(D\).
  • Constraint \(y=\frac12x^2\) reduces to one variable.
  • Geometric check: shortest segment is perpendicular to the tangent.

PART 2 — Strategy & Tips (roadmap only)

  1. Parameterize curve: \(Q(x)=(x,\frac12x^2)\).
  2. Write squared distance \(D^2(x)=(x-4)^2+(\frac12x^2-1)^2\).
  3. Differentiate \(D^2(x)\) and set \(D^2{}'(x)=0\).
  4. Solve for critical points (expect a cubic, but simplifies here).
  5. Confirm minimum via \(D^2{}''(x)>0\) or global behavior.
  6. Compute \(y=\frac12x^2\) for final coordinates.
Common mistakes:
  • Minimizing \(D\) instead of \(D^2\) (harder).
  • Forgetting chain rule on \((\frac12x^2-1)^2\).

PART 3 — Full Solution

Intuition: the nearest point occurs where the line from \(P\) hits the parabola along the normal direction (perpendicular to the tangent).
A point on the parabola is \(Q(x)=\left(x,\frac12x^2\right)\). Squared distance to \(P(4,1)\): \[ D^2(x)=(x-4)^2+\left(\frac12x^2-1\right)^2. \] Differentiate: \[ \frac{d}{dx}(x-4)^2 = 2(x-4), \] and with \(u(x)=\frac12x^2-1\), \(u'(x)=x\), \[ \frac{d}{dx}\,u^2 = 2u\,u' = 2\left(\frac12x^2-1\right)x. \] So \[ (D^2)' = 2(x-4)+2x\left(\frac12x^2-1\right). \] Set \((D^2)'=0\) and divide by 2: \[ (x-4)+x\left(\frac12x^2-1\right)=0 \Rightarrow x-4+\frac12x^3-x=0 \Rightarrow \frac12x^3=4 \Rightarrow x^3=8 \Rightarrow x=2. \] Then \[ y=\frac12x^2=\frac12(4)=2. \]
Final Answer
\[ \boxed{Q=(2,2)} \]
Sanity checks:
  • \(D^2{}''(x)=3x^2\Rightarrow D^2{}''(2)=12>0\) ⇒ minimum.
  • Tangent slope at \(x=2\): \(y' = x = 2\). Line \(PQ\) slope is \(\frac{2-1}{2-4}=-\frac12\). Product \(2\cdot(-\frac12)=-1\) ⇒ perpendicular. ✅
