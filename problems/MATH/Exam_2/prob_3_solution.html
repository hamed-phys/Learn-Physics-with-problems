Problem 3 — Quadratic approximation of \(f(x)=\ln(\sin x)\) at \(x=\pi/2\)

Compute the 2nd-degree Taylor polynomial \(T_2(x)\) about \(a=\pi/2\).

file: prob3.html

PART 1 — Problem Analysis (no solving yet)

Restate: Find \(T_2(x)=f(a)+f'(a)(x-a)+\frac{f''(a)}{2}(x-a)^2\) for \(f(x)=\ln(\sin x)\), \(a=\pi/2\).
Given: \(f(x)=\ln(\sin x)\), \(a=\pi/2\).
Unknown: \(T_2(x)\).
Need: \(f(a),f'(a),f''(a)\).
Why Taylor: local modeling near a point (physics-style small-perturbation approximation). Near \(x=\pi/2\), \(\sin x\approx 1\) so \(f(x)\) is smooth and easy to expand.

PART 2 — Strategy & Tips (roadmap only)

  1. Set \(a=\pi/2\).
  2. Compute \(f(a)=\ln(\sin a)\).
  3. Differentiate: \(f'(x)=\frac{(\sin x)'}{\sin x}\).
  4. Evaluate \(f'(a)\) using trig values at \(\pi/2\).
  5. Differentiate again to find \(f''(x)\) (use \(\frac{d}{dx}\cot x=-\csc^2 x\)).
  6. Evaluate \(f''(a)\).
  7. Assemble \(T_2(x)\).
Common mistakes:
  • Forgetting \((\ln u)' = u'/u\).
  • Dropping minus sign in \(\frac{d}{dx}\cot x\).
  • Ignoring domain: need \(\sin x>0\) (true near \(\pi/2\)).

PART 3 — Full Solution

Intuition: \(\sin x\) peaks at \(\pi/2\), so \(f'(a)=0\) and curvature should be negative.
Let \(a=\pi/2\).

1) Compute \(f(a)\) \[ f(a)=\ln(\sin(\pi/2))=\ln(1)=0. \] 2) Compute \(f'(x)\) \[ f'(x)=\frac{\cos x}{\sin x}=\cot x. \] So \[ f'(a)=\cot(\pi/2)=0. \] 3) Compute \(f''(x)\) \[ f''(x)=\frac{d}{dx}(\cot x)=-\csc^2 x = -\frac{1}{\sin^2 x}. \] Hence \[ f''(a)=-\csc^2(\pi/2)=-1. \] 4) Assemble \(T_2(x)\) \[ T_2(x)=f(a)+f'(a)(x-a)+\frac{f''(a)}{2}(x-a)^2 = 0 + 0\cdot(x-\pi/2) + \frac{-1}{2}(x-\pi/2)^2. \]
Final Answer
\[ \boxed{T_2(x)= -\frac12\left(x-\frac{\pi}{2}\right)^2} \]
Sanity checks:
  • \(T_2(\pi/2)=0\) matches \(f(\pi/2)=0\).
  • For \(x\neq \pi/2\) nearby, \(T_2(x)<0\), matching \(\sin x<1 \Rightarrow \ln(\sin x)<0\).
