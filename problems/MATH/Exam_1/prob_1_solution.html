Problem 1 — Derivative from First Principles

Calculus
Limits
First Principles

The Problem

Given \( y(x)=\dfrac{1}{x^2}\) for \(x>0\), compute the derivative using only:

\[ y'(x)=\lim_{h\to 0}\frac{y(x+h)-y(x)}{h}. \]

PART 1 — Problem Analysis (no solving yet)

Restate

You must find \(y'(x)\) for \(y=1/x^2\) using the limit definition (not the power rule).

Given
  • \(y(x)=1/x^2\)
  • Domain: \(x>0\)
Unknown
  • \(y'(x)\)
Principles (why)
  • Derivative definition: instantaneous rate of change = limit of average rate.
  • Algebra of fractions + factoring: needed to simplify the difference quotient before taking the limit.
Approaches
  • ✅ First-principles limit + algebra (required)
  • Power rule (fast but not allowed)

PART 2 — Strategy & Tips (roadmap only)

  • Write \(y'(x)=\lim_{h\to 0}\dfrac{y(x+h)-y(x)}{h}\).
  • Substitute \(y(x)=1/x^2\) into \(y(x+h)\) and \(y(x)\).
  • Combine the numerator into a single fraction (common denominator).
  • Factor to cancel an \(h\) safely.
  • Take the limit \(h\to 0\).
  • Do sanity checks: sign, behavior as \(x\to\infty\) and \(x\to 0^+\), units.
Common mistakes
  • Plugging \(h=0\) too early → \(0/0\).
  • Incorrect common denominator when subtracting fractions.
  • Ignoring domain \(x>0\) (important near \(x=0\)).

PART 3 — Full Solution

Physical intuition: \(1/x^2\) is an inverse-square-type decay. As \(x\) increases, the function decreases, so \(y'(x)\) should be negative.

\[ y'(x)=\lim_{h\to 0}\frac{y(x+h)-y(x)}{h} =\lim_{h\to 0}\frac{\frac{1}{(x+h)^2}-\frac{1}{x^2}}{h}. \]

Combine the fractions in the numerator using the common denominator \(x^2(x+h)^2\):

\[ \frac{1}{(x+h)^2}-\frac{1}{x^2} =\frac{x^2-(x+h)^2}{x^2(x+h)^2}. \]

Expand \((x+h)^2=x^2+2xh+h^2\), then simplify:

\[ x^2-(x+h)^2=x^2-(x^2+2xh+h^2)=-2xh-h^2=-h(2x+h). \]

Substitute back and cancel \(h\neq 0\) (then take the limit):

\[ y'(x)=\lim_{h\to 0}\frac{-h(2x+h)}{h\,x^2(x+h)^2} =\lim_{h\to 0}\frac{-(2x+h)}{x^2(x+h)^2}. \]

Now take \(h\to 0\):

\[ y'(x)=\frac{-2x}{x^2\cdot x^2}=-\frac{2}{x^3}. \]
Final Answer: \[ \boxed{\,\frac{dy}{dx}=-\frac{2}{x^3}\quad (x>0)\,} \]
Sanity checks
  • Sign: negative, consistent with decreasing \(1/x^2\).
  • Limits: as \(x\to\infty\), \(-2/x^3\to 0\) (flattens). As \(x\to 0^+\), \(-2/x^3\to -\infty\) (very steep).
  • Units: if \(x\sim L\), then \(y\sim L^{-2}\), so \(dy/dx\sim L^{-3}\) matches \(-2/x^3\).
