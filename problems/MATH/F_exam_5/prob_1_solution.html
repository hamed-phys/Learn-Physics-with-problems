Problem 1 (25 points): Compute the antiderivative

Use integration by parts to evaluate the indefinite integral:

∫ x³ e^{−x²} dx

PART 1 — Problem Analysis (no solving yet)

Understand → Plan → Execute

We want an antiderivative of a polynomial × Gaussian form. This structure is extremely common in physics (e.g., Maxwell–Boltzmann factors, diffusion kernels, quantum harmonic oscillator).

1) Restate the problem

  • Compute F(x) such that F′(x) = x³ e^{−x²}.
  • The instruction explicitly asks to use integration by parts.

2) Given / Unknowns / Goal

  • Given: integrand \(x^3 e^{-x^2}\)
  • Unknown: antiderivative \(F(x)\)
  • Find: closed-form expression + constant of integration \(C\)

3) Relevant tools and why

  • Substitution fits because the exponent is \(-x^2\), whose derivative is proportional to \(x\).
  • Integration by parts fits because after substitution we get a polynomial in the new variable times an exponential.
  • Product + chain rule will verify the result by differentiation.

4) Candidate approaches

  • (A) Substitute \(u=x^2\), then do parts on \(\int u e^{-u}du\): clean and standard.
  • (B) Parts directly in \(x\): doable, but bookkeeping is messier due to \(\frac{d}{dx}e^{-x^2}=-2xe^{-x^2}\).
  • (C) Pattern matching with derivatives of \((\text{poly})e^{-x^2}\): fast if experienced, less systematic for beginners.
✅ Best choice

Use u = x² first to simplify the Gaussian, then apply integration by parts. This honors the instruction while keeping the algebra minimal and transparent.

PART 2 — Strategy & Tips (roadmap only)

⚠ Common mistakes

Forgetting the factor 2 in du = 2x dx, sign mistakes in \(\int e^{−u}du = -e^{−u}\), or choosing the wrong part as “u” in integration by parts.

PART 3 — Full Solution

intuition → derivation → checks

Physical intuition

Products like xⁿ e^{−x²} are “Gaussian-weighted polynomials.” Substituting \(u=x^2\) converts the Gaussian into a simple exponential \(e^{-u}\), and the remaining polynomial factor becomes manageable using integration by parts.

Step-by-step derivation

1) Substitution:
Let u = x²  ⇒  du = 2x dx  ⇒  x dx = (1/2) du
Also x² = u.

Rewrite x³ dx = x²·x dx = u · (1/2) du.

Therefore:
∫ x³ e^{−x²} dx = (1/2) ∫ u e^{−u} du.
We simplified the Gaussian exponent; now we have “polynomial × exponential” in u.
2) Integration by parts on I = ∫ u e^{−u} du:

Choose:
a = u        ⇒ da = du
db = e^{−u}du ⇒ b = ∫ e^{−u}du = −e^{−u}

Apply: ∫ a db = ab − ∫ b da

I = u(−e^{−u}) − ∫ (−e^{−u}) du
  = −u e^{−u} + ∫ e^{−u} du
  = −u e^{−u} − e^{−u} + C
  = −(u+1)e^{−u} + C.
Key sign: ∫ e^{−u} du = −e^{−u}.
3) Put it together and substitute back:

∫ x³ e^{−x²} dx
= (1/2) I
= (1/2) [ −(u+1)e^{−u} ] + C
= −(1/2)(u+1)e^{−u} + C.

Now u = x²:
= −(1/2)(x²+1)e^{−x²} + C.
Back-substitution returns the answer in the original variable x.

✅ Final Result

∫ x³ e^{−x²} dx = −½ (x² + 1) e^{−x²} + C

Sanity checks

  • Differentiate to verify: If \(F(x)= -\tfrac12 (x^2+1)e^{-x^2}\), then by product rule + chain rule, \(F'(x)=x^3 e^{-x^2}\). ✅
  • Units: Exponentials require a dimensionless argument, so \(x^2\) must be dimensionless (or \(x\) is scaled). The result has consistent “integral-of-\(x^3\)” scaling. ✅
  • Limits: As \(x\to\infty\), \(e^{-x^2}\to 0\), so the antiderivative tends to a constant (here it tends to 0 relative to C). Near \(x=0\), the integrand behaves like \(x^3\), matching the derivative behavior. ✅
  • Physics interpretation: This is a Gaussian moment-type integral, common in probability/thermal physics and wavefunctions. ✅
