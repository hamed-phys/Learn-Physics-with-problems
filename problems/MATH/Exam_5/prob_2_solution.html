Problem 2 — Integrate using Polynomial Division

Compute the antiderivative of a rational function by first reducing it into standard, integrable pieces. This is the same simplification mindset used in physics: isolate the “regular background” part and the “singular” part, then integrate each cleanly.

Prompt

Use polynomial division to compute the indefinite integral (for \(x>1\)):

\[ \int \frac{x^3-1}{(x-1)^2}\,dx \]
Goal Find \(F(x)\) with \(F'(x)=\dfrac{x^3-1}{(x-1)^2}\)

PART 1 — Problem Analysis (no solving yet)

We clarify what’s being asked, what’s given, and which methods are valid—before doing any algebraic work.

Analysis

Restate the problem

  • Compute an antiderivative for \(\dfrac{x^3-1}{(x-1)^2}\) under the condition \(x>1\).
  • Use polynomial division as the primary simplification technique.

Given / Unknown / Find

  • Given: integrand \(\dfrac{x^3-1}{(x-1)^2}\), condition \(x>1\).
  • Unknown: antiderivative \(F(x)\).
  • Find: \(F(x)+C\) such that \(F'(x)=\dfrac{x^3-1}{(x-1)^2}\).

Relevant “physics-style” principles

  • This is an algebra + calculus problem (no direct physical law needed).
  • The condition \(x>1\) matters because it ensures \((x-1)>0\), so \(\ln|x-1|=\ln(x-1)\) (real-valued, clean expression).
  • Method mindset: decompose the expression into “easy modes” (polynomial part) + “singular mode” (\(\sim 1/(x-1)\)).
Approach A
Polynomial division → split into standard integrals Best when numerator degree ≥ denominator degree. Produces a polynomial plus a simple remainder term. This is exactly what the prompt asks for.
Approach B
Substitute \(u=x-1\) Can simplify the denominator, but you’ll still need algebraic manipulation; often not as direct as division here.
Approach C
Partial fractions (after simplification) Works well if the expression is already proper or cleanly factored. Still, division is the fastest first move.
Best choice: Polynomial division first, because the rational function is “improper” (degree 3 over degree 2), and division directly produces integrable pieces with minimal overhead.

  • Log absolute value: remember \(\int \frac{1}{x-1}\,dx=\ln|x-1|+C\). Then use \(x>1\) to simplify.
  • Forgetting +C: always include the constant of integration.

PART 3 — Full Solution

We start with intuition, then execute polynomial division, integrate term-by-term, and finish with sanity checks.

Solution

Physical intuition (calculus intuition)

  • The denominator \((x-1)^2\) signals a potential singularity at \(x=1\).
  • The numerator \(x^3-1\) also vanishes at \(x=1\), so the singularity weakens after cancellation.
  • After simplification, we expect a polynomial “background” plus a \(\frac{1}{x-1}\) term whose integral is logarithmic.
▾ Step 1 — Polynomial division setup Rewrite the denominator as a polynomial.
Expand the denominator
\[ (x-1)^2 = x^2 - 2x + 1 \]

Because \(\deg(x^3-1)=3\) and \(\deg(x^2-2x+1)=2\), we perform long division.

▾ Step 2 — Long division (showing the key subtractions) Extract polynomial + remainder.
Long division core moves

Divide \(x^3-1\) by \(x^2-2x+1\).

\[ \frac{x^3-1}{x^2-2x+1} \]

Leading term: \(\frac{x^3}{x^2}=x\). Multiply back and subtract:

\[ x(x^2-2x+1)=x^3-2x^2+x \] \[ (x^3-1)-(x^3-2x^2+x)=2x^2-x-1 \]

Next term: \(\frac{2x^2}{x^2}=2\). Multiply back and subtract:

\[ 2(x^2-2x+1)=2x^2-4x+2 \] \[ (2x^2-x-1)-(2x^2-4x+2)=3x-3 \]
Division result
\[ x^3-1 = (x+2)(x-1)^2 + (3x-3) \]
▾ Step 3 — Simplify the integrand completely Turn it into standard integrals.
Divide both sides by \((x-1)^2\)
\[ \frac{x^3-1}{(x-1)^2} = x+2 + \frac{3x-3}{(x-1)^2} \] Factor \(3x-3=3(x-1)\): \[ \frac{3x-3}{(x-1)^2}=\frac{3(x-1)}{(x-1)^2}=\frac{3}{x-1} \] Therefore: \[ \frac{x^3-1}{(x-1)^2} = x+2+\frac{3}{x-1} \]
▾ Step 4 — Integrate term-by-term Apply standard rules.
Split the integral
\[ \int \frac{x^3-1}{(x-1)^2}\,dx = \int\left(x+2+\frac{3}{x-1}\right)\,dx \] \[ = \int x\,dx + \int 2\,dx + 3\int\frac{1}{x-1}\,dx \]
Compute each piece
\[ \int x\,dx = \frac{x^2}{2},\qquad \int 2\,dx = 2x,\qquad \int\frac{1}{x-1}\,dx = \ln|x-1| \] So: \[ F(x)=\frac{x^2}{2}+2x+3\ln|x-1|+C \]
Use \(x>1\) to simplify the log
Since \(x>1\Rightarrow x-1>0\), we have \(\ln|x-1|=\ln(x-1)\). Hence: \[ F(x)=\frac{x^2}{2}+2x+3\ln(x-1)+C \quad (x>1) \]

✅ Final Result

\[ \boxed{ \int \frac{x^{3}-1}{(x-1)^{2}}\,dx =\frac{x^{2}}{2}+2x+3\ln(x-1)+C \quad (x>1) } \]

Sanity checks

  • Units: No physical units are specified; the expression is dimensionless. Log applies to a dimensionless quantity \((x-1)\) under the given domain.
  • Large-\(x\) behavior: \(\frac{x^3-1}{(x-1)^2}\sim \frac{x^3}{x^2}=x\). Derivative of \(\frac{x^2}{2}\) is \(x\) — consistent.
  • Near \(x=1^+\): simplified integrand \(x+2+\frac{3}{x-1}\) is dominated by \(\frac{3}{x-1}\); integral behaves like \(3\ln(x-1)\to -\infty\), consistent with a logarithmic divergence.
  • Physics-style interpretation: decomposition into a smooth “background” (\(x+2\)) plus a singular “mode” (\(\sim 1/(x-1)\)) naturally yields a polynomial term plus a log term.
▸ Optional verification (differentiate to confirm) Quick correctness check.
Differentiate \(F(x)\)
\[ F(x)=\frac{x^2}{2}+2x+3\ln(x-1)+C \] \[ F'(x)=x+2+\frac{3}{x-1} \] Combine: \[ x+2+\frac{3}{x-1} = \frac{(x+2)(x-1)+3}{x-1} = \frac{x^2+x-2+3}{x-1} = \frac{x^2+x+1}{x-1} \] And from division we had: \[ \frac{x^3-1}{(x-1)^2} = x+2+\frac{3}{x-1} \] so it matches exactly.
