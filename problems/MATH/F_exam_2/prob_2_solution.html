Principles (why they apply)

  • Second derivative gives concavity and inflection points (sign change).

Common mistakes

  • Forgetting the domain \(x>0\) (no \(y\)-intercept and you only approach \(0\) from the right).
  • Missing that \(x^{-3/2}\) and \(x^{-5/2}\) are always positive for \(x>0\): only the linear factor sets the sign.

PART 3 ‚Äî Full Solution

step-by-step
Physical intuition

The term \( \frac{1}{\sqrt{x}} \) dominates as \(x\to 0^+\) and diverges to \(+\infty\), while \( \sqrt{x} \) dominates as \(x\to\infty\) and also grows without bound. So the graph must be high near \(0^+\), then dip, then rise again‚Äîsuggesting one minimum.

(a) Vertical asymptote(s)

A vertical asymptote occurs where the function becomes unbounded. Since \(x>0\), the only boundary point to test is \(x\to 0^+\).

\[ \lim_{x\to 0^+}\left(\sqrt{x}+\frac{1}{\sqrt{x}}\right) = 0 + \infty = +\infty. \]

Answer
\(\boxed{x=0}\)
(b) Horizontal asymptote(s)

A horizontal asymptote would require a finite limit as \(x\to\infty\).

\[ \lim_{x\to\infty}\left(\sqrt{x}+\frac{1}{\sqrt{x}}\right) = \infty + 0 = \infty. \]

Answer
\(\boxed{\text{none}}\)
Compute \(f'(x)\) carefully

Rewrite using exponents: \( f(x)=x^{1/2}+x^{-1/2}\). Differentiate term-by-term using \( \frac{d}{dx}x^n = nx^{n-1}\).

\[ f'(x)=\frac12 x^{-1/2}-\frac12 x^{-3/2}. \]

Factor the common positive term \( \frac12 x^{-3/2}\) (positive for \(x>0\)):

\[ f'(x)=\frac12 x^{-3/2}(x-1). \]

(c) Where \(f'(x)\) is positive/negative/zero

For \(x>0\), \( \frac12 x^{-3/2} >0\). Therefore the sign of \(f'(x)\) is the sign of \(x-1\).

Interval / PointSign of \(x-1\)Sign of \(f'(x)\)Meaning
\((0,1)\)negative\(f'(x)<0\)decreasing
\(x=1\)\(0\)\(f'(1)=0\)critical point
\((1,\infty)\)positive\(f'(x)>0\)increasing
Answer
\(\boxed{f'(x)<0\text{ on }(0,1),\ f'(1)=0,\ f'(x)>0\text{ on }(1,\infty)}\)
(d) Local maximum coordinates

A local maximum would require \(f'\) to change from \(+\) to \(-\). But \(f'(x)\) changes from \(-\) to \(+\) at \(x=1\), which indicates a minimum, not a maximum.

Answer
\(\boxed{\text{none}}\)
(e) Local minimum coordinates

Since \(f'(x)\) changes from negative to positive at \(x=1\), \(x=1\) is a local minimum. Compute \(f(1)\):

\[ f(1)=\sqrt{1}+\frac{1}{\sqrt{1}}=1+1=2. \]

Answer
\(\boxed{(1,2)}\)
Compute \(f''(x)\) carefully

Differentiate \( f'(x)=\frac12 x^{-1/2}-\frac12 x^{-3/2}\) term-by-term:

\[ f''(x)=\frac12\left(-\frac12 x^{-3/2}\right) -\frac12\left(-\frac32 x^{-5/2}\right) = -\frac14 x^{-3/2}+\frac34 x^{-5/2}. \]

Factor \( \frac14 x^{-5/2} \) (positive for \(x>0\)):

\[ f''(x)=\frac14 x^{-5/2}(3-x). \]

(g) Where \(f''(x)\) is positive/negative/zero

For \(x>0\), \( \frac14 x^{-5/2}>0\), so the sign comes from \(3-x\).

Interval / PointSign of \(3-x\)Sign of \(f''(x)\)Concavity
\((0,3)\)positive\(f''(x)>0\)concave up
\(x=3\)\(0\)\(f''(3)=0\)candidate inflection
\((3,\infty)\)negative\(f''(x)<0\)concave down
Answer
\(\boxed{f''>0\text{ on }(0,3),\ f''(3)=0,\ f''<0\text{ on }(3,\infty)}\)
(f) Inflection point(s)

Since concavity changes from up to down at \(x=3\), this is an inflection point. Compute the \(y\)-value:

\[ f(3)=\sqrt3+\frac{1}{\sqrt3} = \sqrt3+\frac{\sqrt3}{3} = \frac{4}{3}\sqrt3. \]

Using \(\sqrt3\approx 1.7\): \(\frac{4}{3}\sqrt3 \approx \frac{4}{3}(1.7)\approx 2.27\).

Answer
\(\boxed{\left(3,\frac{4}{3}\sqrt3\right)\approx(3,2.27)}\)
(h) Sketch instructions (what your graph must show)
  • Domain: only \(x>0\). No graph left of the \(y\)-axis.
  • Vertical asymptote: \(x=0\) with \(f(x)\to +\infty\) as \(x\to0^+\).
  • Monotonicity: decreasing on \((0,1)\), increasing on \((1,\infty)\).
  • Local minimum: at \((1,2)\).
  • Concavity: concave up on \((0,3)\), concave down on \((3,\infty)\).
  • Inflection: at \(x=3\), \(y=\frac{4}{3}\sqrt3\approx2.27\).
  • No horizontal asymptote: \(f(x)\to\infty\) as \(x\to\infty\).
Sanity checks (units, limits, interpretation)

Units: \( \sqrt{x}\) and \(1/\sqrt{x}\) can be added only if \(x\) is dimensionless (typical in calculus). If \(x\) had physical units, the expression would be inconsistent‚Äîso treat \(x\) as unitless.

Limits: \(x\to0^+ \Rightarrow f(x)\to\infty\) (matches vertical asymptote), \(x\to\infty \Rightarrow f(x)\sim \sqrt{x}\to\infty\) (no horizontal asymptote).

Interpretation: one term dominates each extreme, forcing one ‚Äúdip‚Äù ‚Üí exactly one minimum (found at \(x=1\)).
