PART 1 Problem Analysis (no solving yet)
Restated task: For each system (A–E), choose generalized coordinates for small oscillations and write the linear equations in the matrix eigenvalue form (A − ω²I)C = 0, with A = M⁻¹K for mass–spring systems (and the circuit analog for the LC ladder). Do not solve for ω or modes.
Given notes/assumptions:
  • Spring ends not attached to masses are fixed.
  • Ignore vertical motion (B).
  • Ignore ring curvature (C) ⇒ treat motion as 1D periodic chain.
  • For the LC ladder (D), write equations in terms of inductor currents.
  • Small angles only (E) ⇒ sinθ ≈ θ, cosθ ≈ 1 − θ²/2.
Given quantities (what you can use) ›
  • Springs: k and 3k as labeled.
  • Masses: m (and one 3m in system A).
  • Lengths: L in pendulum-based systems (B, E).
  • Gravity: g enters via small-angle pendulum restoring terms (B, E).
  • LC ladder: inductors L and capacitors C in system D.
Unknowns (what we must produce) ›
  • Generalized coordinate vector C for each system.
  • Matrix A such that (A − ω²I)C = 0.
Why these principles apply ›
  • Small oscillations: linearization yields constant-coefficient ODEs.
  • Newton + Hooke: mass–spring forces depend linearly on relative extensions.
  • Pendulum approx: for small angles, horizontal restoring acceleration is (g/L)x.
  • Circuit analog: inductors/capacitors store energy and yield coupled linear dynamics.
Candidate approaches (and best choice) ›
  • Newton + Hooke for A, C (and B with gravity term): fastest to assemble K.
  • Lagrange energy method for E: clean mass matrix with coupling.
  • KCL/KVL for D: naturally uses inductor currents as requested.

Best: Use the natural method for each system (Newton / Lagrange / circuit laws).

PART 2 Strategy & Tips (roadmap only)

Minimal roadmap (5–10 steps)

  1. Choose coordinates qᵢ (xᵢ, θᵢ, or iᵢ).
    Goal: independent small motions.
  2. Write linear element laws (springs/pends/LC).
    Tool: Hooke, sinθ≈θ, v=L di/dt, i=C dv/dt.
  3. Write EOM (Newton or Lagrange) or KCL/KVL (circuit).
    Goal: M q̈ + K q = 0 (or analog).
  4. Read off matrices M and K from coefficients.
    Tool: compare term-by-term.
  5. Convert to eigenvalue form using q(t)=C e^{iωt}.
    Tool: q̈ = −ω² q ⇒ (K − ω²M)C=0 ⇒ (M⁻¹K − ω²I)C=0.

Common mistakes & quick tips

  • Sign errors: consistently use spring extension (xᵢ − xⱼ).
  • Fixed ends: wall springs add +k xᵢ to stiffness.
  • Pendulum: horizontal restoring force is −m(g/L)x, not −mgx.
  • Ring periodicity: include coupling between 4 and 1.
  • LC ladder: keep current directions consistent across KCL equations.
PART 3 Full Solution
System A — 3 masses, 4 springs, fixed ends ›

Physical intuition: each spring provides a restoring force proportional to relative extension; end masses also feel wall springs.

Coordinates: C = [x₁, x₂, x₃]ᵀ, where xᵢ are horizontal displacements of masses (m, 3m, m).

Derivation (linearized Newton + Hooke)

Mass 1: m ẍ₁ = −k x₁ − k(x₁ − x₂) = −2k x₁ + k x₂
Mass 2: 3m ẍ₂ = −k(x₂ − x₁) − k(x₂ − x₃) = k x₁ − 2k x₂ + k x₃
Mass 3: m ẍ₃ = −k(x₃ − x₂) − k x₃ = k x₂ − 2k x₃

So M ẍ + K x = 0 with:

M = diag(m, 3m, m)
K = k [[ 2, -1, 0], [-1, 2, -1], [ 0, -1, 2]]

Final matrix form: assume x(t)=C e^{iωt} ⇒ ẍ=−ω²x

(K − ω² M)C = 0 ⟺ (A − ω²I)C = 0, A = M⁻¹K
A = [ 2k/m -k/m 0 ] [ -k/(3m) 2k/(3m) -k/(3m)] [ 0 -k/m 2k/m ]
Sanity checks: Units: k/m → s⁻². If the middle mass becomes very large (3m→∞), its row weakens (slower response), as expected.
System B — 3 pendula + springs, horizontal motion only ›

Physical intuition: each bob experiences gravity-based restoring force like an effective spring (g/L), plus coupling via real springs.

Coordinates: C = [x₁, x₂, x₃]ᵀ, small horizontal displacements, with small-angle relation x≈Lθ.

Derivation (linearized)

Small-angle pendulum gives F_g ≈ −m(g/L)x.

m ẍ₁ = −m(g/L)x₁ − k(x₁ − x₂)
m ẍ₂ = −m(g/L)x₂ − k(x₂ − x₁) − 3k(x₂ − x₃)
m ẍ₃ = −m(g/L)x₃ − 3k(x₃ − x₂)

Thus M = mI and:

K = [ m g/L + k -k 0 ] [ -k m g/L + 4k -3k] [ 0 -3k m g/L + 3k]
(A − ω²I)C = 0, with A = (1/m)K:
A = [ g/L + k/m -k/m 0 ] [ -k/m g/L + 4k/m -3k/m] [ 0 -3k/m g/L + 3k/m]
Sanity checks: If k→0, then A→(g/L)I ⇒ three independent pendula with ω²=g/L.
System C — 4 masses on a ring, springs k and 3k (periodic) ›

Physical intuition: ignoring curvature turns the ring into a periodic 1D chain; each mass couples to two neighbors.

Coordinates: C = [x₁, x₂, x₃, x₄]ᵀ, tangential displacements along the ring. Springs: (1–2)=k, (2–3)=3k, (3–4)=k, (4–1)=3k.

Stiffness assembly rule

  • Diagonal Kᵢᵢ = sum of spring constants attached to mass i.
  • Off-diagonal Kᵢⱼ = −(spring constant between i and j), else 0.
  • Periodic coupling: include (4,1).
M = m I₄ K = [ 4k -k 0 -3k] [ -k 4k -3k 0 ] [ 0 -3k 4k -k ] [-3k 0 -k 4k ]
(A − ω²I)C = 0, A = (1/m)K:
A = (k/m) * [ 4 -1 0 -3] [-1 4 -3 0] [ 0 -3 4 -1] [-3 0 -1 4]
Sanity checks: Units k/m → s⁻². If 3k→k (all equal springs), K becomes the uniform periodic chain stiffness.
System D — LC ladder (3 inductors, 4 capacitors), use inductor currents ›

Physical intuition: inductors behave like inertial elements; capacitors provide restoring behavior. The ladder couples adjacent inductor currents through shared node voltages.

Coordinates (requested): C = [i₁, i₂, i₃]ᵀ, currents through the three inductors (left→right).

Write KCL at capacitor nodes (to ground)

Let node voltages be v₁…v₄. Capacitor current: i_C = C v̇.

C v̇₁ = −i₁
C v̇₂ = i₁ − i₂
C v̇₃ = i₂ − i₃
C v̇₄ = i₃

Inductor voltage relations

v₁ − v₂ = L i̇₁
v₂ − v₃ = L i̇₂
v₃ − v₄ = L i̇₃

Eliminate node voltages to get current-only equations

Differentiate the inductor relations and substitute v̇’s from KCL:

ï₁ + (1/LC)(2 i₁ − i₂) = 0
ï₂ + (1/LC)(−i₁ + 2 i₂ − i₃) = 0
ï₃ + (1/LC)(−i₂ + 2 i₃) = 0
Assume i(t)=C e^{iωt} ⇒ ï=−ω² i, so (A − ω²I)C=0 with:
A = (1/LC) * [ 2 -1 0] [-1 2 -1] [ 0 -1 2]
Sanity checks: 1/(LC) has units s⁻². If C→∞, then A→0 ⇒ ω→0 (no effective restoring, as expected).
System E — 3-link serial pendulum (3 masses), small-angle oscillations ›

Physical intuition: lower masses move when upper links move ⇒ kinetic energy is coupled; gravity provides restoring torques.

Coordinates: C = [θ₁, θ₂, θ₃]ᵀ, small angles from vertical for each link (top, middle, bottom).

Step 1: Small-angle kinematics (horizontal positions)

x₁ = L θ₁
x₂ = L(θ₁ + θ₂)
x₃ = L(θ₁ + θ₂ + θ₃)

Step 2: Kinetic energy ⇒ mass matrix

Using ẋ’s, the kinetic energy is:

T = ½ m (ẋ₁² + ẋ₂² + ẋ₃²) = ½ θ̇ᵀ M θ̇
M = m L² * [3 2 1] [2 2 1] [1 1 1]

Step 3: Potential energy ⇒ stiffness matrix

Small-angle gravitational potential expands to:

V = ½ θᵀ K θ,
K = m g L * diag(3, 2, 1)

Step 4: Lagrange’s equations ⇒ matrix EOM

M θ̈ + K θ = 0 ⇒ (K − ω² M)C = 0 ⇒ (A − ω² I)C = 0, with A = M⁻¹K
Final A (simplified):
A = (g/L) * [ 3 -2 0] [-3 4 -1] [ 0 -2 2]
Sanity checks: g/L has units s⁻². If g→0, A→0 ⇒ ω→0 (no restoring). Strong coupling appears in upper rows because θ₁ influences all masses.
Σ Final Summary (A–E)

Every system is expressed in the compact eigenvalue form (A − ω²I)C = 0, where C is the coordinate amplitude vector and A has units of s⁻².

A — mass–spring (m,3m,m)

C = [x₁,x₂,x₃]ᵀ
A = [ 2k/m -k/m 0 ] [ -k/(3m) 2k/(3m) -k/(3m)] [ 0 -k/m 2k/m ]

B — pendula + springs (horizontal only)

C = [x₁,x₂,x₃]ᵀ
A = [ g/L + k/m -k/m 0 ] [ -k/m g/L + 4k/m -3k/m] [ 0 -3k/m g/L + 3k/m]

C — 4-mass periodic ring

C = [x₁,x₂,x₃,x₄]ᵀ
A = (k/m) * [ 4 -1 0 -3] [-1 4 -3 0] [ 0 -3 4 -1] [-3 0 -1 4]

D — LC ladder (currents in inductors)

C = [i₁,i₂,i₃]ᵀ
A = (1/LC) * [ 2 -1 0] [-1 2 -1] [ 0 -1 2]

E — 3-link serial pendulum (angles)

C = [θ₁,θ₂,θ₃]ᵀ
A = (g/L) * [ 3 -2 0] [-3 4 -1] [ 0 -2 2]

Tip: If your instructor prefers the generalized eigenvalue form, you can equivalently present (K − ω²M)C = 0 for the mechanical systems, where A = M⁻¹K.
