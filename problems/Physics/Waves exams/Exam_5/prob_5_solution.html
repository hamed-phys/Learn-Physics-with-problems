Fixed–Fixed String Struck at the Center: Modes, Excitation, and the Snapshot at \(t=L/(2c)\)

A string is initially flat, \(y(x,0)=0\), but receives a localized velocity kick: \(+v_0\) on \([L-a,L)\) and \(-v_0\) on \([L,L+a)\). We find the first modes, the excited modal amplitude \(A_n\), the lowest unexcited mode, and sketch the displacement at \(t=\frac{L}{2}\sqrt{\frac{\mu}{T}}=\frac{L}{2c}\).

P1 PART 1 — Problem Analysis (no solving yet)

Restatement. A taut string of length \(2L\) is clamped at both ends. Initially it is straight: \(y(x,0)=0\). At \(t=0\), it is struck near the midpoint \(x=L\), giving \(\dot y(x,0)=+v_0\) on the left half of the struck region \([L-a,L)\) and \(\dot y(x,0)=-v_0\) on the right half \([L,L+a)\), with \(a\ll L\). Tasks: sketch the first three normal modes, compute the \(n\)-th mode amplitude, find the lowest unexcited mode, and sketch \(y(x,t)\) at \(t=L/(2c)\).

1 Given

  • Length: \(2L\), density: \(\mu\), tension: \(T\), speed: \(c=\sqrt{T/\mu}\)
  • \(y(x,0)=0\)
  • \[ \dot y(x,0)= \begin{cases} +v_0, & L-a\le x

2 Unknowns / goals

  • Mode shapes for \(n=1,2,3\)
  • Amplitude \(A_n\) of the \(n\)-th normal mode after the strike
  • Lowest unexcited mode
  • Displacement sketch at \(t=\frac{L}{2}\sqrt{\frac{\mu}{T}}=\frac{L}{2c}\)

3 Physics principles (why)

  • Wave equation: small transverse motion of a taut string: \(\;y_{tt}=c^2y_{xx}\), \(c=\sqrt{T/\mu}\).
  • Fixed ends: clamped boundaries impose \(y(0,t)=y(2L,t)=0\).
  • Normal modes: linear PDE + fixed BC → sine eigenfunctions with orthogonality, enabling projection.
  • Finite propagation: at \(t=L/(2c)\), the disturbance travels \(L/2\), so no reflections yet.

4 Approaches + best choice

  1. Normal modes for (5a, 5b): directly yields mode amplitudes.
  2. Traveling-wave reasoning for (5c): simplest pre-reflection snapshot.
Best approach: mode expansion for amplitudes, traveling-wave logic for the snapshot.

P2 PART 2 — Strategy & Tips (roadmap only)

  1. Write PDE + BCs. Tool: \(y_{tt}=c^2y_{xx}\), \(y(0,t)=y(2L,t)=0\).
  2. Write eigenmodes/frequencies. Tool: \(\phi_n=\sin(\frac{n\pi x}{2L})\), \(\omega_n=c\frac{n\pi}{2L}\).
  3. Expand \(y\) in modes. Tool: \(y=\sum q_n(t)\phi_n(x)\).
  4. Use \(y(x,0)=0\) to get \(q_n(0)=0\).
  5. Project initial velocity. Tool: orthogonality \(\int_0^{2L}\phi_n\phi_m dx=L\delta_{nm}\).
  6. Solve \(q_n''+\omega_n^2q_n=0\). Tool: \(q_n(t)=\frac{\dot q_n(0)}{\omega_n}\sin(\omega_n t)\).
  7. Identify unexcited modes. Tool: symmetry factor \(\cos(n\pi/2)\).
  8. Sketch at \(t=L/(2c)\). Tool: no-reflection argument → two triangular pulses.
Common mistakes.
  • Using the wrong length in sines (must be \(2L\)).
  • Forgetting the normalization \(\int_0^{2L}\sin^2(\cdot)\,dx=L\).
  • Using \(\cos(\omega_n t)\) when \(q_n(0)=0\) (it should be \(\sin(\omega_n t)\)).

P3 PART 3 — Full Solution

Intuition. The kick is antisymmetric about the midpoint: left gets \(+v_0\), right gets \(-v_0\). That symmetry removes coupling to odd modes and launches two opposite-sign pulses moving outward.

5a First three normal modes (fixed ends)

Fixed ends enforce \(y(0,t)=y(2L,t)=0\). The eigenfunctions are:

\[ \phi_n(x)=\sin\!\left(\frac{n\pi x}{2L}\right),\qquad \omega_n=c\frac{n\pi}{2L},\qquad c=\sqrt{\frac{T}{\mu}}. \]
  • \(n=1\): one half-wave (single hump), maximum at \(x=L\).
  • \(n=2\): two humps with a node at \(x=L\).
  • \(n=3\): three half-waves (two interior nodes).
Qualitative mode sketches for Figure 8 (exact shapes are sine functions).

5b Amplitude of the \(n\)-th mode & lowest unexcited mode

Expand: \[ y(x,t)=\sum_{n=1}^{\infty} q_n(t)\sin\!\left(\frac{n\pi x}{2L}\right),\quad \omega_n=c\frac{n\pi}{2L}. \] Since \(y(x,0)=0\), we have \(q_n(0)=0\) for all \(n\).

\[ \dot q_n(0)=\frac{1}{L}\int_{0}^{2L}\dot y(x,0)\sin\!\left(\frac{n\pi x}{2L}\right)\,dx. \]

Let \(k_n=\frac{n\pi}{2L}\). Insert the piecewise velocity:

\[ \dot q_n(0)=\frac{v_0}{L}\left[\int_{L-a}^{L}\sin(k_nx)\,dx-\int_{L}^{L+a}\sin(k_nx)\,dx\right]. \]

Compute integrals (\(\int\sin(kx)\,dx=-\cos(kx)/k\)) and simplify using \(\cos(k(L-a))+\cos(k(L+a))=2\cos(kL)\cos(ka)\) and \(\cos(ka)-1=-2\sin^2(ka/2)\). Result:

\[ \dot q_n(0)= -\frac{4v_0}{k_nL}\cos(k_nL)\sin^2\!\left(\frac{k_na}{2}\right). \]

Since \(k_nL=\frac{n\pi}{2}\), we have \(\cos(k_nL)=\cos\!\left(\frac{n\pi}{2}\right)\). With \(q_n(0)=0\), the modal motion is: \[ q_n(t)=\frac{\dot q_n(0)}{\omega_n}\sin(\omega_nt). \] So the (signed) amplitude \(A_n=\dot q_n(0)/\omega_n\) is:

Final amplitude and mode motion

\[ \boxed{ A_n= -\frac{16Lv_0}{c\,n^2\pi^2}\, \cos\!\left(\frac{n\pi}{2}\right)\, \sin^2\!\left(\frac{n\pi a}{4L}\right) } \] \[ \boxed{ q_n(t)=A_n\sin(\omega_nt),\qquad \omega_n=c\frac{n\pi}{2L}. } \]
Lowest unexcited mode. \(\cos\!\left(\frac{n\pi}{2}\right)=0\) for odd \(n\), so all odd modes are unexcited. Therefore: \[ \boxed{n=1\text{ is the lowest unexcited mode.}} \]

5c Snapshot at \(t=\dfrac{L}{2}\sqrt{\dfrac{\mu}{T}}=\dfrac{L}{2c}\)

At this time \(ct=L/2\), the disturbance has not reached the ends (they are distance \(L\) from the midpoint). So the shape is just the outward-propagating response: a positive triangular pulse centered at \(x=L/2\) and a negative triangular pulse centered at \(x=3L/2\), each of width \(2a\).

Piecewise displacement at \(t=L/(2c)\)

\[ \boxed{ y(x,t)= \begin{cases} \displaystyle \frac{v_0}{2c}\left(x-\frac{L}{2}+a\right), & \frac{L}{2}-a\le x\le \frac{L}{2}\\[10pt] \displaystyle \frac{v_0}{2c}\left(\frac{L}{2}+a-x\right), & \frac{L}{2}\le x\le \frac{L}{2}+a\\[10pt] 0, & \frac{L}{2}+a\le x\le \frac{3L}{2}-a\\[10pt] \displaystyle \frac{v_0}{2c}\left(\frac{3L}{2}-a-x\right), & \frac{3L}{2}-a\le x\le \frac{3L}{2}\\[10pt] \displaystyle -\frac{v_0}{2c}\left(\frac{3L}{2}+a-x\right), & \frac{3L}{2}\le x\le \frac{3L}{2}+a\\[10pt] 0,& \text{elsewhere} \end{cases}} \]
Peak magnitude. \[ \boxed{ y\!\left(\frac{L}{2},t\right)=+\frac{v_0a}{2c},\quad y\!\left(\frac{3L}{2},t\right)=-\frac{v_0a}{2c} } \] Using \(1/c=\sqrt{\mu/T}\), \[ \boxed{\text{peak}=\frac{v_0a}{2}\sqrt{\frac{\mu}{T}}.} \]

✓ Sanity checks

U Units

  • \(c=\sqrt{T/\mu}\) has units \(m/s\).
  • \(\frac{v_0a}{2c}\) has units \(m\).
  • \(A_n=\dot q_n(0)/\omega_n\) has units \(m\).

L Limits

  • \(a\to0\Rightarrow\) peak \(\propto a\to0\) (vanishes).
  • \(v_0\to0\Rightarrow\) no motion.
  • Odd \(n\Rightarrow \cos(n\pi/2)=0\Rightarrow\) no excitation (symmetry).

P Interpretation

  • Antisymmetric velocity kick about the midpoint suppresses odd modes (including the fundamental).
  • Early time behavior: two opposite-sign pulses moving outward, later reflected into a standing-wave superposition.
Final boxed takeaway. \[ \boxed{ \phi_n(x)=\sin\!\left(\frac{n\pi x}{2L}\right),\quad \omega_n=c\frac{n\pi}{2L},\quad A_n= -\frac{16Lv_0}{c\,n^2\pi^2} \cos\!\left(\frac{n\pi}{2}\right) \sin^2\!\left(\frac{n\pi a}{4L}\right),\quad n_{\min,\text{unexcited}}=1.} \]
