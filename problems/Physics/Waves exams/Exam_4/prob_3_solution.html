Problem 3 ‚Äî Deformed String (Free‚ÄìFixed End) ¬∑ Full Structured Solution

A stretched string of length L has a free end at \(x=0\) (massless sliding ring) and a fixed end at \(x=L\). At \(t=0\), the string is released from rest with a narrow rectangular pulse of height \(H\) and width \(D\) centered at \(x_0=L/4\). We find the full normal-mode spectrum, coefficients, time evolution, and the half-period shape.

Mixed BCs: \(y_x(0,t)=0\), \(y(L,t)=0\) Wave speed: \(c=\sqrt{T/\mu}\) Released from rest: only \(\cos(\omega_mt)\)

PART 1 ‚Äî Problem Analysis (no solving yet)

Conceptual setup

Restatement

A string of length \(L\), tension \(T\), density \(\mu\) vibrates transversely. At \(x=L\) it is fixed: \(y(L,t)=0\). At \(x=0\) it is attached to a massless ring that can move vertically on a frictionless rod, so the end is transverse-force-free: \(y_x(0,t)=0\). Initially, \(y(x,0)\) is a narrow rectangular pulse (height \(H\), width \(D\ll L\)) centered at \(x_0=L/4\), and the string is released from rest: \(y_t(x,0)=0\).

Given / Unknowns / Goals

  • Given: \(L, T, \mu, H, D, x_0=L/4\), and \(y_t(x,0)=0\).
  • Unknowns: normal modes \(X_m(x)\), wavenumbers \(k_m\), frequencies \(\omega_m\), periods \(\tau_m\), coefficients \(A_m\).
  • Find: (a)‚Äì(e) exactly as stated: full spectrum, expansion, zeros, time series, and half-period shape.

Physics principles (and why)

  • Wave equation for a stretched string: small transverse motion satisfies \(y_{tt}=c^2y_{xx}\) with \(c=\sqrt{T/\mu}\).
  • Boundary conditions from constraints: fixed end gives \(y=0\); free end gives \(y_x=0\) (no transverse force).
  • Eigenfunction expansion: separation of variables yields orthogonal normal modes; initial shape is projected onto them.
Candidate approaches (compare & choose) Expand
  • Normal-mode/Fourier method: systematic for (a)‚Äì(d); directly produces coefficients via orthogonality.
  • Traveling-wave reflection method: great intuition for (e); trickier for explicit \(A_m\).
  • Standing-wave wavelength counting: fast spectrum derivation; still needs projection for coefficients.

Chosen approach: normal modes + projection, because it cleanly solves all parts and makes (e) immediate.

PART 2 ‚Äî Strategy & Tips (roadmap only)

Execution plan
  1. Compute wave speed \(c=\sqrt{T/\mu}\).
  2. Separate variables \(y=XG\) to get \(X''+k^2X=0\), \(G''+\omega^2G=0\), \(\omega=ck\).
  3. Apply BCs \(X'(0)=0\), \(X(L)=0\) to quantize \(k=k_m\) and determine \(X_m(x)\).
  4. Write mode time dependence and use \(y_t(x,0)=0\) to keep only \(\cos(\omega_mt)\) terms.
  5. Expand initial shape \(y(x,0)=\sum A_mX_m(x)\).
  6. Project to find \(A_m\) using orthogonality; evaluate integrals for the rectangle explicitly.
  7. Identify any zeros by analyzing closed-form factors in \(A_m\).
  8. Evaluate \(t=\tau_1/2\) by inserting into \(\cos(\omega_mt)\) for all modes (or reflection logic).
Common mistakes:
  • Using \(y(0,t)=0\) instead of \(y_x(0,t)=0\).
  • Wrong mode indexing: mixed BCs \(\Rightarrow\) half-integer quantization.
  • Forgetting normalization \(\int_0^L X_m^2\,dx\).
  • Keeping sine-in-time terms despite \(y_t(x,0)=0\).

PART 3 ‚Äî Full Solution

Derivations + results
Physical intuition: Free end \(\Rightarrow\) slope zero (antinode-type); fixed end \(\Rightarrow\) displacement zero (node). This ‚Äúfree‚Äìfixed‚Äù combination shifts the allowed standing waves compared to fixed‚Äìfixed: the spectrum uses odd half-integers. The initial pulse is decomposed into these standing waves, and since the string is released from rest, each mode oscillates as \(\cos(\omega_m t)\).

(a) Functional form, \(k_m\), \(\omega_m\), and \(\tau_m\)

Spectrum

Wave equation (small transverse oscillations):

\[ y_{tt}=c^2 y_{xx},\qquad c=\sqrt{\frac{T}{\mu}}. \]

Boundary conditions:

\[ y(L,t)=0 \quad(\text{fixed end}),\qquad y_x(0,t)=0 \quad(\text{free end / ring}). \]

Separation \(y=X(x)G(t)\) gives:

\[ X''+k^2X=0,\qquad G''+\omega^2G=0,\qquad \omega=ck. \]

General spatial solution: \(X=A\cos(kx)+B\sin(kx)\). Apply \(X'(0)=0\):

\[ X'(x)=-Ak\sin(kx)+Bk\cos(kx)\;\Rightarrow\;X'(0)=Bk=0\;\Rightarrow\;B=0, \] \[ \Rightarrow\; X(x)=A\cos(kx). \]

Apply \(X(L)=0\):

\[ \cos(kL)=0 \;\Rightarrow\; kL=\left(m+\tfrac12\right)\pi,\quad m=0,1,2,\dots \]

Result (a)

\[ X_m(x)=\cos(k_m x), \qquad k_m=\frac{(2m+1)\pi}{2L},\quad m=0,1,2,\dots \]
\[ \omega_m=ck_m=\sqrt{\frac{T}{\mu}}\;\frac{(2m+1)\pi}{2L}, \qquad \tau_m=\frac{2\pi}{\omega_m}=\frac{4L}{(2m+1)}\sqrt{\frac{\mu}{T}}. \]

(b) Coefficients \(A_m\) for the rectangular initial pulse

Projection

Initial displacement (rectangle of height \(H\), width \(D\), centered at \(x_0=L/4\)): let \(a=x_0-\tfrac{D}{2}\), \(b=x_0+\tfrac{D}{2}\).

\[ y(x,0)= \begin{cases} H,& a\le x\le b,\\ 0,& \text{otherwise}, \end{cases} \qquad y_t(x,0)=0. \]

Expand the initial shape in the orthogonal eigenfunctions:

\[ y(x,0)=\sum_{m=0}^{\infty} A_m\cos(k_m x). \]

Orthogonality (for these mixed-BC cosine modes):

\[ \int_{0}^{L}\cos(k_m x)\cos(k_n x)\,dx=\frac{L}{2}\delta_{mn}. \]

So the projection formula is:

\[ A_m=\frac{\int_0^L y(x,0)\cos(k_m x)\,dx}{\int_0^L\cos^2(k_m x)\,dx} =\frac{2}{L}\int_0^L y(x,0)\cos(k_m x)\,dx =\frac{2H}{L}\int_a^b \cos(k_m x)\,dx. \]

Evaluate the integral and simplify using \(\sin u-\sin v=2\cos\!\big(\tfrac{u+v}{2}\big)\sin\!\big(\tfrac{u-v}{2}\big)\):

\[ A_m=\frac{2H}{L}\left[\frac{\sin(k_m x)}{k_m}\right]_a^b =\frac{2H}{Lk_m}\left(\sin(k_m b)-\sin(k_m a)\right) =\frac{4H}{Lk_m}\cos(k_m x_0)\sin\!\left(k_m\frac{D}{2}\right). \]

Insert \(x_0=L/4\) and \(k_m=\frac{(2m+1)\pi}{2L}\):

Result (b)

\[ A_m= \frac{8H}{(2m+1)\pi}\, \cos\!\left(\frac{(2m+1)\pi}{8}\right)\, \sin\!\left(\frac{(2m+1)\pi D}{4L}\right), \qquad m=0,1,2,\dots \]

(c) Which coefficients are exactly zero?

Selection rules

From

\[ A_m\propto \cos\!\left(\frac{(2m+1)\pi}{8}\right)\, \sin\!\left(\frac{(2m+1)\pi D}{4L}\right), \]

we can have \(A_m=0\) only if one of these factors is zero.

Check the cosine factor Expand

\(\cos\!\left(\frac{(2m+1)\pi}{8}\right)=0\) would require \(\frac{(2m+1)\pi}{8}=\frac{\pi}{2}+n\pi\Rightarrow 2m+1=8n+4\Rightarrow m=4n+\frac{3}{2}\), which is never an integer. So the pulse center \(x_0=L/4\) does not eliminate any integer \(m\) modes.

Check the sine factor Expand

\(\sin\!\left(\frac{(2m+1)\pi D}{4L}\right)=0\) occurs only for special tuned widths:

\[ \frac{(2m+1)\pi D}{4L}=p\pi \;\Rightarrow\; D=\frac{4Lp}{2m+1},\quad p\in\mathbb{Z}. \]

For a generic \(D\ll L\), this does not hold exactly, so typically no coefficients vanish.

Result (c)

In general (generic \(D\)): \(\boxed{\text{no }A_m=0}\).
Only if \(D=\dfrac{4Lp}{2m+1}\) for some integers \((p,m)\) does that particular \(A_m\) vanish.

(d) Full time-dependent Fourier (normal-mode) decomposition

Dynamics

The general modal time dependence is \(C_m\cos(\omega_m t)+D_m\sin(\omega_m t)\). Since \(y_t(x,0)=0\), all \(D_m=0\).

Result (d)

\[ y(x,t)=\sum_{m=0}^{\infty} A_m\,\cos(k_m x)\,\cos(\omega_m t), \]
\[ k_m=\frac{(2m+1)\pi}{2L},\qquad \omega_m=\sqrt{\frac{T}{\mu}}\;\frac{(2m+1)\pi}{2L}, \]
\[ A_m= \frac{8H}{(2m+1)\pi}\, \cos\!\left(\frac{(2m+1)\pi}{8}\right)\, \sin\!\left(\frac{(2m+1)\pi D}{4L}\right). \]

(e) Shape at \(t=\tau_1/2\) (half-period of the lowest mode)

Quick insight

The lowest mode is \(m=0\), so

\[ \tau_1=\tau_0=\frac{4L}{c}\quad\Rightarrow\quad \frac{\tau_1}{2}=\frac{2L}{c}. \]

Evaluate the phase for any mode \(m\):

\[ \omega_m\left(\frac{2L}{c}\right)=ck_m\left(\frac{2L}{c}\right)=2Lk_m =2L\cdot\frac{(2m+1)\pi}{2L}=(2m+1)\pi. \]

Thus, for every mode \(m\),

\[ \cos\!\left(\omega_m\frac{\tau_1}{2}\right)=\cos\big((2m+1)\pi\big)=-1. \]

Result (e)

\[ y\!\left(x,\frac{\tau_1}{2}\right) =\sum_{m=0}^{\infty}A_m\cos(k_m x)\left[-1\right] =-\,y(x,0). \]

The entire initial shape flips sign: the rectangular ‚Äúbump‚Äù becomes an identical rectangular ‚Äúdip‚Äù at the same location.

Sanity checks

Units ¬∑ limits ¬∑ meaning

Units

  • \(c=\sqrt{T/\mu}\): \(T\) is N \(=\mathrm{kg\,m/s^2}\), \(\mu=\mathrm{kg/m}\) \(\Rightarrow c=\mathrm{m/s}\).
  • \(k_m\sim 1/L\) and \(\omega_m=ck_m\sim 1/\mathrm{s}\).
  • \(A_m\) carries units of length (same as \(H\)); trig factors are dimensionless.

Limiting cases

  • If \(D\to 0\): \(\sin(k_m D/2)\approx k_m D/2\) so coefficients scale \(\propto HD/L\), consistent with a narrow localized displacement.
  • Higher \(m\): coefficients are generally suppressed by \(1/(2m+1)\) and oscillatory trig factors, as expected.
  • Half-period result: all modes pick up a phase \(\pi\) because the spectrum is odd multiples of the fundamental‚Äîso the whole shape inverts.
