Concept Map

Big picture (physics-style intuition)

Treat w(x,y) like a “landscape” (potential surface). Critical points are places where the slope is zero. The Hessian tells whether the landscape curves down (max), up (min), or mixes directions (saddle). For constrained regions (like the first quadrant), maxima can move to boundaries.

Problem 3 (given function and tasks)

w(x,y) = -3x² - 4xy - y² - 12y + 16x
What is given?
  • A quadratic scalar function w(x,y).
  • A constraint region for part (b): x ≥ 0, y ≥ 0.
What is unknown?
  • Critical point(s) (solutions to ∇w = 0).
  • Type of each critical point (max/min/saddle).
  • The point in the first quadrant where w is largest.
Key tools (and why they apply)
  • Gradient test: stationary points satisfy ∇w = 0.
  • Hessian test: classifies stationary points via curvature.
  • Constrained optimization: check interior + boundaries + behavior at infinity (region is unbounded).

  • New variables u and v defined from x, y.

What must be found?
  • Expressions for wx, wy in terms of wu, wv.

  • Multivariable chain rule: wx = wuux + wvvx, etc.

PART 3 — Full Solution

Step-by-step derivation

Problem 3(a): Find the critical points and classify them

Physical intuition: A critical point is where the “slope” of the surface w(x,y) is zero—an equilibrium. Classification tells whether it’s a peak (max), valley (min), or pass (saddle).

1) Compute the gradient ∇w

w(x,y) = -3x² - 4xy - y² - 12y + 16x wₓ = ∂w/∂x = -6x - 4y + 16 wᵧ = ∂w/∂y = -4x - 2y - 12

2) Solve ∇w = 0

Set wₓ = 0 and wᵧ = 0: -6x - 4y + 16 = 0 ⇒ 3x + 2y = 8 -4x - 2y - 12 = 0 ⇒ 2x + y = -6 From 2x + y = -6 ⇒ y = -6 - 2x Substitute into 3x + 2y = 8: 3x + 2(-6 - 2x) = 8 3x - 12 - 4x = 8 -x = 20 ⇒ x = -20 y = -6 - 2(-20) = 34

3) Hessian test for classification

Second derivatives: wₓₓ = ∂²w/∂x² = -6 wᵧᵧ = ∂²w/∂y² = -2 wₓᵧ = ∂²w/(∂x∂y) = -4 Discriminant: D = wₓₓ wᵧᵧ - (wₓᵧ)² = (-6)(-2) - (-4)² = 12 - 16 = -4 < 0

Since D < 0, the critical point is a saddle point.

Final result (3a)

Critical point: (-20, 34) Type: Saddle point (D = -4 < 0)

Problem 3(b): Largest w in the first quadrant (x ≥ 0, y ≥ 0)

Idea: The only stationary point is (-20,34), but it is not feasible because x<0. So the maximum in the first quadrant must occur on the boundary (x=0 or y=0) or at the corner (0,0), and we must also verify behavior as (x,y) → ∞.

1) Boundary x = 0, with y ≥ 0

w(0,y) = -y² - 12y This parabola opens downward. Its vertex is at d/dy(-y² - 12y) = -2y - 12 = 0 ⇒ y = -6 (not allowed since y ≥ 0). So on y ≥ 0 the maximum occurs at y = 0: w(0,0) = 0

2) Boundary y = 0, with x ≥ 0

w(x,0) = -3x² + 16x d/dx(-3x² + 16x) = -6x + 16 = 0 ⇒ x = 16/6 = 8/3 Evaluate: w(8/3, 0) = -3(64/9) + 16(8/3) = -64/3 + 128/3 = 64/3

3) Global-maximum justification (unbounded region)

As x,y → ∞ with x ≥ 0, y ≥ 0, the quadratic terms dominate: w(x,y) ≈ -(3x² + 4xy + y²) → -∞ So w cannot blow up to +∞; it goes to -∞ far away. Thus the global maximum must occur on the boundary/corner (since no feasible interior critical point exists).

Final result (3b)

Maximum on x ≥ 0, y ≥ 0 occurs at: (8/3, 0) Maximum value: w_max = 64/3
U Units

If x and y carry units, all terms in w are consistent (quadratic/linear in x,y with matching coefficients). The maximizing point has the same units as x and y.

L Limiting behavior

In the first quadrant, w → −∞ as x,y grow, so a finite global maximum is expected — consistent.

Φ Interpretation

The unconstrained stationary point is a saddle outside the feasible region, so the constrained maximum “slides” to the boundary y=0 at x=8/3.

Problem 4(a): Express wx and wy in terms of wu, wv

Physical intuition: w depends on x and y only through u and v. So changes in x or y affect w by changing u and v. The chain rule measures that “flow” through u and v.

1) Compute ux, uy, vx, vy

u = y/x ⇒ uₓ = ∂/∂x(y/x) = -y/x², uᵧ = ∂/∂y(y/x) = 1/x v = x² + y² ⇒ vₓ = 2x, vᵧ = 2y

2) Apply the multivariable chain rule

wₓ = wᵤ uₓ + wᵥ vₓ = wᵤ(-y/x²) + wᵥ(2x) wᵧ = wᵤ uᵧ + wᵥ vᵧ = wᵤ(1/x) + wᵥ(2y)

Final result (4a)

w_x = -(y/x^2) w_u + (2x) w_v w_y = (1/x) w_u + (2y) w_v

Problem 4(b): Express xwx + ywy in terms of wu, wv

Compute xwx + ywy and simplify

xwₓ + ywᵧ = x[ wᵤ(-y/x²) + wᵥ(2x) ] + y[ wᵤ(1/x) + wᵥ(2y) ] Group wᵤ terms: x·wᵤ(-y/x²) + y·wᵤ(1/x) = wᵤ[ -(xy/x²) + (y/x) ] = wᵤ[ -(y/x) + (y/x) ] = 0 Group wᵥ terms: x·(2x)wᵥ + y·(2y)wᵥ = 2(x² + y²) wᵥ = 2v wᵥ

Coefficients as functions of (u,v): the coefficient of wu is 0 (a constant function), and the coefficient of wv is 2v.

Final result (4b)

x w_x + y w_y = 0·w_u + (2v)·w_v ⇒ x w_x + y w_y = 2v w_v

Problem 4(c): Find xwx + ywy if w = v5

If w = v^5, then w_v = ∂/∂v (v^5) = 5v^4 From (4b): xw_x + yw_y = 2v w_v ⇒ xw_x + yw_y = 2v(5v^4) = 10v^5 = 10(x^2 + y^2)^5

Final result (4c)

x w_x + y w_y = 10 v^5 = 10 (x^2 + y^2)^5 (Equivalently: x w_x + y w_y = 10 w)
U Units

If v has units of length², then v⁵ has units length¹⁰. The operator x∂ₓ + y∂ᵧ preserves w’s units, so getting 10v⁵ is consistent.

L Limiting case

If v → 0 (near the origin), then xwₓ + ywᵧ → 0, matching 10v⁵ → 0.

Φ Interpretation

x∂ₓ + y∂ᵧ measures response under scaling (x,y)→(λx,λy). Since v scales like λ², v⁵ scales like λ¹⁰, producing the factor 10.
