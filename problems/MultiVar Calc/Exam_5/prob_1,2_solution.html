PART 1 — Problem Analysis no solving yet

Problem restatement what is asked?

(1) For a uniform triangular lamina with vertices \((-2,0)\), \((0,1)\), \((2,0)\) and density \(\delta=1\), write an explicit integral setup (integrands + limits) for \(\bar y\) without evaluating, and find \(\bar x\).

(2) For the unit disk \(x^2+y^2\le 1\) with density equal to distance from the \(y\)-axis (\(\rho=|x|\)), compute the polar moment of inertia about the origin (the \(z\)-axis).

(3) For \(\vec F=\langle ax^2y+y^3+1,\; 2x^3+bxy^2+2\rangle\), determine when \(\vec F\) is conservative, find a potential \(f\) with \(\vec F=\nabla f\), and evaluate \(\int_C \vec F\cdot d\vec r\) along \(x=e^t\cos t,\; y=e^t\sin t,\;0\le t\le\pi\).

Given / Unknowns inventory

  • Given: Triangle vertices \((-2,0),(0,1),(2,0)\), uniform density \(\delta=1\).
  • Given: Unit disk \(x^2+y^2\le 1\), density \(\rho(x,y)=|x|\).
  • Given: Vector field \(P=ax^2y+y^3+1\), \(Q=2x^3+bxy^2+2\).
  • Given: Curve \(C: (x,y)=(e^t\cos t,e^t\sin t)\), \(t\in[0,\pi]\).
  • Unknowns: Integral form for \(\bar y\); value of \(\bar x\); polar MOI \(I\); constants \(a,b\); potential \(f\); line integral \(\int_C \vec F\cdot d\vec r\).

Relevant laws & why they apply core physics/math

  • Center of mass of a lamina: \[ \bar x=\frac{1}{M}\iint_R x\,\delta\,dA,\qquad \bar y=\frac{1}{M}\iint_R y\,\delta\,dA,\qquad M=\iint_R \delta\,dA. \] Applies because the object is a 2D region with known density.
  • Polar moment of inertia about the origin: \[ I=\iint_R r^2\,\rho(x,y)\,dA,\quad r^2=x^2+y^2. \] Applies because MOI weights each mass element by squared distance to the axis.
  • Conservative field test in \(\mathbb{R}^2\): if domain is simply connected, then \[ \vec F=\langle P,Q\rangle \text{ conservative } \iff \frac{\partial P}{\partial y}=\frac{\partial Q}{\partial x}. \] Then \(\vec F=\nabla f\) and the line integral is path-independent: \[ \int_C \vec F\cdot d\vec r=f(B)-f(A). \]
Approaches considered (and best choices) expand
  • Triangle centroid: piecewise Cartesian bounds + symmetry (clean and explicit).
  • Disk MOI: polar coordinates (region becomes \(0\le r\le 1\); handles \(r^2\) naturally).
  • Line integral: if conservative, use endpoints via potential (fast and robust).

PART 2 — Strategy & Tips roadmap only

Minimal step-by-step plan 5–10 steps

  1. Triangle boundaries: derive the two edge lines and piecewise \(y\)-limits. Tool: line equations from two points.
  2. Set up \(\bar y\): write \(\bar y=\dfrac{\iint_R y\,dA}{\iint_R dA}\) using explicit piecewise integrals. Tool: double integrals in Cartesian coordinates.
  3. Find \(\bar x\): invoke symmetry about the \(y\)-axis. Tool: symmetry/oddness.
  4. MOI in polar: use \(\rho=|x|=r|\cos\theta|\), \(r^2=x^2+y^2\), \(dA=r\,dr\,d\theta\). Tool: polar coordinate transformation.
  5. Conservative test: compute \(\partial P/\partial y\), \(\partial Q/\partial x\), match coefficients \(\Rightarrow a,b\). Tool: equality for all \(x,y\).
  6. Find potential: integrate \(P\) w.r.t. \(x\) to get \(f\), then use \(f_y=Q\) to determine missing \(g(y)\). Tool: gradient matching.
  7. Line integral: compute endpoints from \(t=0,\pi\), evaluate \(f(B)-f(A)\). Tool: fundamental theorem for line integrals.

Common mistakes & quick tips avoid traps

  • Triangle: forgetting the region is piecewise in \(x\) (left vs right edge).
  • Disk: dropping the absolute value: \(\rho=|x|=r|\cos\theta|\).
  • Conservative: mixing up partial derivatives or matching only at one point (must hold for all \(x,y\)).
  • Potential: forgetting the “+ \(g(y)\)” after integrating with respect to \(x\).
  • Line integral: re-parametrizing unnecessarily once a potential exists.
Tip: When an integrand separates into a product of an \(r\)-part and a \(\theta\)-part (as in the MOI), compute them independently for speed and clarity.

PART 3 — Full Solution intuition → derivation → checks

Physical intuition start here

  • Center of mass: uniform density means the centroid is the balance point. This triangle is mirror-symmetric about the \(y\)-axis, so \(\bar x\) must be \(0\).
  • Polar MOI: measures how much “mass is spread out” from the origin. Since \(\rho=|x|\) increases toward the left/right edges, the disk is heavier near \(|x|=1\), increasing inertia compared to uniform density.
  • Conservative field: if \(\vec F=\nabla f\), then the work along any path depends only on endpoints: \(\int_C \vec F\cdot d\vec r=f(B)-f(A)\).

1) Center of mass of the triangle (\(\delta=1\))

Triangle vertices: \((-2,0)\), \((0,1)\), \((2,0)\). Base lies on \(y=0\). Top vertex at \((0,1)\).

Step 1 — Boundary lines
Line from \((-2,0)\) to \((0,1)\): slope \(\frac{1-0}{0-(-2)}=\frac12\), hence \[ y=\frac12(x+2),\qquad -2\le x\le 0. \] Line from \((0,1)\) to \((2,0)\): slope \(\frac{0-1}{2-0}=-\frac12\), hence \[ y=\frac12(2-x),\qquad 0\le x\le 2. \] The region uses \(0\le y\le\) (top boundary).
Step 2 — Mass and \(\bar y\) as a ratio
With uniform density \(\delta=1\), \[ M=\iint_R 1\,dA,\qquad \bar y=\frac{1}{M}\iint_R y\,dA. \] Using the piecewise \(x\)-intervals: \[ \boxed{ \bar y= \frac{ \displaystyle \int_{-2}^{0}\int_{0}^{(x+2)/2} y\,dy\,dx + \int_{0}^{2}\int_{0}^{(2-x)/2} y\,dy\,dx }{ \displaystyle \int_{-2}^{0}\int_{0}^{(x+2)/2} 1\,dy\,dx + \int_{0}^{2}\int_{0}^{(2-x)/2} 1\,dy\,dx }. } \] This is the requested explicit integral setup (no evaluation).
Step 3 — \(\bar x\) from symmetry
The region is symmetric about the \(y\)-axis and the density is uniform. Thus the centroid must lie on the \(y\)-axis: \[ \boxed{\bar x=0.} \]
Sanity check: \(\bar x\) has units of length and must be zero by mirror symmetry.

2) Polar moment of inertia of the unit disk \(\rho=|x|\)

Step 1 — Interpret density
Distance from the \(y\)-axis (\(x=0\)) is \(|x|\). So \[ \rho(x,y)=|x|. \]
Step 2 — Polar moment of inertia about the origin
For a lamina in the \(xy\)-plane, the polar moment about the \(z\)-axis through the origin is \[ I=\iint_{x^2+y^2\le 1} (x^2+y^2)\,\rho(x,y)\,dA. \] Let \(r^2=x^2+y^2\). Then \(I=\iint r^2\,\rho\,dA\).
Step 3 — Convert to polar coordinates
Use \[ x=r\cos\theta,\quad y=r\sin\theta,\quad dA=r\,dr\,d\theta,\quad 0\le r\le 1,\ 0\le\theta\le 2\pi. \] Then \[ \rho=|x|=|r\cos\theta|=r|\cos\theta|. \] Therefore \[ I=\int_0^{2\pi}\int_0^1 r^2\,(r|\cos\theta|)\,(r\,dr\,d\theta) =\int_0^{2\pi}\int_0^1 r^4|\cos\theta|\,dr\,d\theta. \]
Step 4 — Separate and evaluate
The integrand is a product of an \(r\)-part and a \(\theta\)-part: \[ I=\left(\int_0^1 r^4\,dr\right)\left(\int_0^{2\pi}|\cos\theta|\,d\theta\right). \] Compute the radial integral: \[ \int_0^1 r^4\,dr=\left[\frac{r^5}{5}\right]_0^1=\frac15. \] For the angular integral, \(|\cos\theta|\) repeats each quadrant: \[ \int_0^{2\pi}|\cos\theta|\,d\theta=4\int_0^{\pi/2}\cos\theta\,d\theta =4\left[\sin\theta\right]_0^{\pi/2}=4. \] Hence \[ \boxed{I=\frac15\cdot 4=\frac45.} \]
Sanity checks:
  • Sign: \(I>0\) since \(r^2\ge0\) and \(\rho\ge0\).
  • Interpretation: density increases with \(|x|\), so more mass sits near the left/right edges, increasing inertia.

3) Conservative field, potential, and line integral \(\vec F=\langle P,Q\rangle\)

\[ P(x,y)=ax^2y+y^3+1,\qquad Q(x,y)=2x^3+bxy^2+2. \]
(a) Find \(a,b\) for conservativeness
In a simply connected domain (here \(\mathbb{R}^2\)), the condition is \[ \frac{\partial P}{\partial y}=\frac{\partial Q}{\partial x}. \] Compute: \[ \frac{\partial P}{\partial y}=ax^2+3y^2,\qquad \frac{\partial Q}{\partial x}=6x^2+by^2. \] Set equal for all \(x,y\): \[ ax^2+3y^2=6x^2+by^2. \] Match coefficients: \[ a=6,\qquad b=3. \] \[ \boxed{a=6,\ b=3.} \]
(b) Find potential \(f\) with \(\vec F=\nabla f\)
With \(a=6,\ b=3\), \[ f_x=P=6x^2y+y^3+1. \] Integrate with respect to \(x\) (treat \(y\) as constant): \[ f(x,y)=\int(6x^2y+y^3+1)\,dx =2x^3y+xy^3+x+g(y), \] where \(g(y)\) is an unknown function of \(y\). Differentiate with respect to \(y\) and set equal to \(Q\): \[ f_y=2x^3+3xy^2+g'(y)\stackrel{!}{=}2x^3+3xy^2+2 \Rightarrow g'(y)=2. \] So \(g(y)=2y+C\). Hence \[ \boxed{f(x,y)=2x^3y+xy^3+x+2y+C.} \]
(c) Compute \(\displaystyle \int_C \vec F\cdot d\vec r\)
Because \(\vec F\) is conservative, \[ \int_C \vec F\cdot d\vec r=f(B)-f(A). \] Curve: \(x=e^t\cos t,\ y=e^t\sin t,\ 0\le t\le\pi\). Endpoints: \[ A=(x(0),y(0))=(1,0),\qquad B=(x(\pi),y(\pi))=(-e^\pi,0). \] Evaluate \(f\) on \(y=0\): \[ f(x,0)=2x^3\cdot 0 + x\cdot 0 + x + 2\cdot 0 + C=x+C. \] Therefore \[ \int_C \vec F\cdot d\vec r = ( -e^\pi + C ) - ( 1 + C ) = -e^\pi - 1. \] \[ \boxed{\displaystyle \int_C \vec F\cdot d\vec r=-(e^\pi+1).} \]
Sanity check: Since along the \(x\)-axis the potential reduces to \(f=x+C\), moving from \(x=1\) to \(x=-e^\pi\) must decrease \(f\) by \(e^\pi+1\), so the work is negative: \(-\,(e^\pi+1)\).
