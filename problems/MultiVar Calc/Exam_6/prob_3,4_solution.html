PART 1 — Problem Analysis (no solving yet)

Concepts: conservative fields • potentials • Green’s theorem
Problem 3
(a) Show \(\mathbf F=(3x^2-6y^2)\hat{\mathbf i}+(-12xy+4y)\hat{\mathbf j}\) is conservative.
(b) Find a potential \(\phi\) with \(\nabla\phi=\mathbf F\).
(c) For \(C:\ x=1+y^3(1-y)^3,\ 0\le y\le 1\), compute \(\displaystyle\int_C \mathbf F\cdot d\mathbf r\).
Problem 4
(a) Write the work around the unit circle CCW once as \(\displaystyle \int_a^b f(t)\,dt\) (do not simplify).
(b) Evaluate the line integral using Green’s theorem.

Given, unknowns, and goals

We write \(\mathbf F=\langle P,Q\rangle\).

Problem 3
\(P(x,y)=3x^2-6y^2,\quad Q(x,y)=-12xy+4y.\)
Curve \(C:\ x(y)=1+y^3(1-y)^3,\ 0\le y\le 1.\)

Problem 4
\(P(x,y)=5x+3y,\quad Q(x,y)=1+\cos y.\)
\(C:\ x^2+y^2=1\) oriented counterclockwise.

Relevant principles & why they apply

  • Conservative test (2D): If \(\partial P/\partial y=\partial Q/\partial x\) everywhere on a simply connected region, \(\mathbf F\) is conservative → work is path independent.
  • Potential function: If conservative, \(\mathbf F=\nabla\phi\) and \(\displaystyle\int_C \mathbf F\cdot d\mathbf r=\phi(B)-\phi(A)\).
  • Green’s theorem: Converts \(\displaystyle\oint_C P\,dx+Q\,dy\) into \(\displaystyle\iint_D(\partial Q/\partial x-\partial P/\partial y)\,dA\), ideal for circular regions.

Approaches (and best choices)

Options:

  1. Direct parameterization of curves & integrate. Works always but can be algebra-heavy (especially Problem 3c).
  2. Conservative shortcut (Problem 3): find \(\phi\) and use endpoints. Cleanest and fastest.
  3. Green’s theorem (Problem 4b): best for closed curves like the unit circle.

Chosen: Use the conservative test + potential method for Problem 3; use parameterization only to write \(f(t)\) in 4a, and Green’s theorem for 4b.

PART 2 — Strategy & Tips (roadmap only)

5–10 steps • no algebra
  1. Test conservativeness for Problem 3.
    Compute \(\partial P/\partial y\) and \(\partial Q/\partial x\); compare.
  2. Construct the potential \(\phi\).
    Integrate \(P\) w.r.t. \(x\): \(\phi=\int P\,dx+g(y)\).
  3. Determine the missing \(y\)-only function.
    Differentiate \(\phi\) w.r.t. \(y\), match \(\phi_y=Q\), solve \(g'(y)\).
  4. Identify endpoints for curve \(C\) (Problem 3c).
    Plug \(y=0\) and \(y=1\) into \(x(y)\).
  5. Compute the work along \(C\) using path independence.
    \(\displaystyle\int_C \mathbf F\cdot d\mathbf r=\phi(B)-\phi(A)\).
  6. For Problem 4a, parameterize the unit circle CCW.
    \((x(t),y(t))=(\cos t,\sin t)\), \(t\in[0,2\pi]\); compute \(dx,dy\).
  7. Write the integrand \(f(t)\) without simplification.
    \(\oint P\,dx+Q\,dy=\int_0^{2\pi}\big(P(x(t),y(t))x'(t)+Q(x(t),y(t))y'(t)\big)\,dt\).
  8. Evaluate Problem 4b via Green’s theorem.
    \(\displaystyle\iint_D(\partial Q/\partial x-\partial P/\partial y)\,dA\) over unit disk.
Common mistakes
  • For Green’s theorem, mixing the sign: CCW uses \(\partial Q/\partial x-\partial P/\partial y\).
  • When integrating \(P\) to get \(\phi\), forgetting the extra \(g(y)\) term.
  • Brute-forcing Problem 3c instead of using \(\phi(B)-\phi(A)\).

PART 3 — Full Solution

intuition • derivations • checks
Physical intuition:
A conservative field behaves like an idealized force derived from a potential energy landscape: the work depends only on the endpoints. That’s why Problem 3c becomes a clean potential difference. For Problem 4, Green’s theorem relates circulation around a closed loop to the “curl density” over the enclosed area.

Problem 3

(a) Show that \(\mathbf F\) is conservative

\[ \mathbf F=\langle P,Q\rangle=\langle 3x^2-6y^2,\ -12xy+4y\rangle. \] Compute cross-partials: \[ \frac{\partial P}{\partial y}=\frac{\partial}{\partial y}(3x^2-6y^2)=-12y, \qquad \frac{\partial Q}{\partial x}=\frac{\partial}{\partial x}(-12xy+4y)=-12y. \] Since \(\frac{\partial P}{\partial y}=\frac{\partial Q}{\partial x}\) everywhere in \(\mathbb R^2\) (simply connected), \[ \boxed{\mathbf F\ \text{is conservative.}} \]

(b) Find a potential function \(\phi\)

We require \(\nabla\phi=\mathbf F\), so \[ \phi_x=P=3x^2-6y^2,\qquad \phi_y=Q=-12xy+4y. \] Integrate \(\phi_x\) with respect to \(x\): \[ \phi(x,y)=\int (3x^2-6y^2)\,dx=x^3-6xy^2+g(y). \] Differentiate with respect to \(y\): \[ \phi_y=\frac{\partial}{\partial y}\big(x^3-6xy^2+g(y)\big)=-12xy+g'(y). \] Match with \(Q\): \[ -12xy+g'(y)=-12xy+4y\ \Rightarrow\ g'(y)=4y\ \Rightarrow\ g(y)=2y^2+C. \] Therefore, \[ \boxed{\phi(x,y)=x^3-6xy^2+2y^2+C.} \]

(c) Compute \(\displaystyle\int_C \mathbf F\cdot d\mathbf r\) for \(x=1+y^3(1-y)^3\), \(0\le y\le 1\)

Because \(\mathbf F\) is conservative, \[ \int_C \mathbf F\cdot d\mathbf r=\phi(B)-\phi(A). \] Determine endpoints via \(y\): \[ y=0:\ x=1+0^3(1-0)^3=1\ \Rightarrow\ A=(1,0), \] \[ y=1:\ x=1+1^3(1-1)^3=1\ \Rightarrow\ B=(1,1). \] Evaluate \(\phi\) (constant \(C\) cancels): \[ \phi(1,1)=1^3-6(1)(1^2)+2(1^2)=1-6+2=-3, \] \[ \phi(1,0)=1^3-6(1)(0)+2(0)=1. \] Hence, \[ \int_C \mathbf F\cdot d\mathbf r = (-3)-1=-4 \quad\Rightarrow\quad \boxed{\int_C \mathbf F\cdot d\mathbf r=-4.} \]
Sanity checks (Problem 3)
  • Path independence: complicated curve, simple result — expected for conservative fields.
  • Units: if \(\mathbf F\) were a force and \(\mathbf r\) a displacement, the integral has units of work.
  • Interpretation: negative work means the field removes energy along the net motion from \((1,0)\) to \((1,1)\).

Problem 4

(a) Write the work as \(\displaystyle\int_a^b f(t)\,dt\) (do not evaluate or simplify)

Given \[ \mathbf F=\langle P,Q\rangle=\langle 5x+3y,\ 1+\cos y\rangle. \] Parameterize the unit circle CCW: \[ x(t)=\cos t,\quad y(t)=\sin t,\quad 0\le t\le 2\pi. \] Then \[ dx = x'(t)\,dt = (-\sin t)\,dt,\qquad dy = y'(t)\,dt = (\cos t)\,dt. \] Substitute into \(\oint P\,dx+Q\,dy\): \[ \oint_C \mathbf F\cdot d\mathbf r =\int_0^{2\pi}\Big(P(x(t),y(t))\,dx + Q(x(t),y(t))\,dy\Big). \] Along the curve, \[ P(x(t),y(t))=5\cos t+3\sin t,\qquad Q(x(t),y(t))=1+\cos(\sin t). \] So the requested form is: \[ \boxed{ \int_{0}^{2\pi} f(t)\,dt \ \text{with}\ f(t)=\big(5\cos t+3\sin t\big)(-\sin t)+\big(1+\cos(\sin t)\big)(\cos t). } \] (Not simplified, as required.)

(b) Evaluate the line integral using Green’s theorem

Green’s theorem (CCW): \[ \oint_C P\,dx+Q\,dy=\iint_D\left(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}\right)\,dA, \] where \(D\) is the unit disk. Compute partial derivatives: \[ \frac{\partial Q}{\partial x}=\frac{\partial}{\partial x}(1+\cos y)=0, \qquad \frac{\partial P}{\partial y}=\frac{\partial}{\partial y}(5x+3y)=3. \] Therefore the integrand is constant: \[ \frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}=0-3=-3. \] So \[ \oint_C P\,dx+Q\,dy=\iint_D (-3)\,dA=-3\cdot \text{Area}(D)=-3\cdot \pi. \] Hence \[ \boxed{\oint_C \mathbf F\cdot d\mathbf r=-3\pi.} \]
Sanity checks (Problem 4)
  • Orientation: CCW corresponds to the Green’s theorem sign used; negative result indicates net clockwise circulation density.
  • Scaling: integrand is constant \(-3\), so answer must be \(-3\times\text{area}\) — here \(-3\pi\).
  • Units: circulation/work units remain consistent (force·length) if \(\mathbf F\) is a force field.
