Peak-to-Peak Output of a Wien-Bridge Oscillator with Diode Amplitude Stabilization

We are given a Wien-bridge oscillator whose adjustable potentiometer is replaced by two fixed resistors: 15 kΩ from node b to the op-amp’s negative input and 10 kΩ from the negative input to ground. The nonlinear branch between nodes a and b contains a 10-kΩ resistor in parallel with two oppositely oriented diodes. Each diode is modeled as a 0.65-V battery in series with 100 Ω.

The task is to find the peak-to-peak amplitude of the output sinusoid. In this circuit the labeled output is taken at node b, and the diode network limits the difference voltage between the op-amp output node a and the external output node b.

Nonlinear feedback Barkhausen condition Piecewise-linear diode model Steady-state sinusoid

Quick Summary

  • This is a Wien-bridge oscillator with amplitude stabilization provided by antiparallel diodes.
  • The key physics idea is that the Wien network gives a gain of 1/3 at the oscillation frequency, so the amplifier must present an effective closed-loop gain of 3.
  • The negative-input divider is fixed by the new resistors, so the op-amp forces a definite relation between the internal node a and the output node b.
  • The mathematical model is a piecewise-linear algebraic steady-state system, not a field PDE problem.
  • The governing relations are: v+=va/3, v-=(10/(15+10))vb=0.4vb, and in linear operation v+=v-.
  • From these, the difference voltage across the diode-resistor branch is x = va-vb = 0.2 vb.
  • At the output peak, one diode conducts, so the nonlinear branch current is i = x/10 kΩ + (x-0.65)/100 Ω.
  • The divider connected to node b draws current i = vb/25 kΩ = x/5 kΩ.
  • Solving gives x ≈ 0.6566 V, so vb,peak = 5x ≈ 3.283 V.
  • Final numeric result: the output sinusoid is approximately 6.57 V peak-to-peak.

PART 0 — Concept Primer (Theory Before Solving)

Core Definitions

  • va(t): op-amp output node voltage, in volts (V).
  • vb(t): external oscillator output node voltage, in volts (V).
  • v+, v-: op-amp noninverting and inverting input voltages, in volts (V).
  • Rs, Rp, Cs, Cp: Wien-bridge lead-lag network components.
  • ω: angular frequency, in rad/s.
  • x = va-vb: voltage across the nonlinear stabilization branch.

Physical Meaning

  • The Wien network selects one frequency where the phase shift is zero.
  • At that frequency its attenuation is 1/3, so the amplifier must compensate with a gain of 3.
  • The diodes act as a soft amplitude limiter: as the signal grows, diode current increases and modifies the feedback relation.
  • This prevents uncontrolled growth and stabilizes the oscillation amplitude.

Key Laws / Principles and Validity Conditions

1) Ideal op-amp constraint in linear operation: when the op-amp is not saturated and the loop is closed, it drives the inputs toward equality:

v+(t) = v-(t)

2) Wien-bridge resonance condition: for the equal-R, equal-C Wien network, at the oscillation frequency:

H(jω0) = v+(t)/va(t) = 1/3 with zero phase shift

3) KCL and resistor laws: currents are determined by node voltages using Ohm’s law and summed using Kirchhoff’s current law.

4) Piecewise-linear diode model: each forward-biased diode behaves as a 0.65-V source in series with 100 Ω; reverse-biased, it is open.

Mathematical Model

This problem is best modeled as a piecewise-linear algebraic steady-state system. The full oscillator startup would involve a linear ODE system for the RC network and feedback loop, but the asked quantity is the steady sinusoidal amplitude after stabilization.

Variables and domain:

  • Time variable: t ∈ ℝ.
  • Voltages: va(t), vb(t), v+(t), v-(t).
  • Steady oscillation assumption: all relevant node voltages are sinusoidal at one angular frequency ω0.

Boundary / operating conditions:

  • No external forcing: self-excited oscillation.
  • At steady state, loop gain magnitude is effectively 1 and total phase shift is 0.
  • At waveform peaks, one diode is forward biased and the other is off.

Because the steady signal is a single-frequency sinusoid, the Wien network can be treated by a phasor relation. That converts the linear RC subproblem from differential form to algebraic form.

Mini Intuition Examples

  • If the loop gain were less than 1, oscillations would decay.
  • If the loop gain were greater than 1, oscillations would grow until nonlinear elements intervene.
  • With no diodes, the amplitude would depend too sensitively on exact gain setting and op-amp saturation limits.

What to Watch For

  • Do not confuse node a (op-amp output) with node b (labeled circuit output).
  • Do not treat the diode branch as a simple short when forward biased; it has both a 0.65-V drop and a 100-Ω series resistance.
  • Do not use the small-signal resistor of the parallel network when the diode is off; the branch is piecewise.
  • At a positive peak, only one diode conducts, not both.
Math Insight

The core equation here is not a PDE but a piecewise-linear constitutive relation: current is a linear function of voltage in each operating region, with switching between regions depending on the sign and magnitude of the branch voltage. Mathematically, this is a nonlinear algebraic system with different affine branches. The steady oscillation amplitude is found by matching the amplifier’s required current-voltage relation to the diode network’s actual one.

PART 0B — Industry & Applied Physics Perspective

Where This Appears in Industry

  • Test and measurement equipment: low-distortion sine-wave generators and calibration sources.
  • Audio electronics: tone generators, filter testing, and analog signal-chain characterization.
  • Industrial instrumentation: excitation sources for bridges, sensors, and impedance-measurement systems.
  • Embedded analog subsystems: clock-like sinusoidal references in narrow-band sensing front ends.

Why Engineers Care

Performance

Amplitude must be stable, startup must be reliable, and distortion must remain low.

Reliability

Nonlinear stabilization avoids hard clipping, which would otherwise stress the amplifier and worsen waveform quality.

Cost & Manufacturability

Replacing an adjustable potentiometer by fixed resistors simplifies assembly, repeatability, and calibration.

Measurement / Validation in Practice

  • Oscilloscope measurement of peak-to-peak amplitude and clipping behavior.
  • Spectrum analyzer or FFT measurement of harmonic distortion.
  • Thermal and tolerance testing to see how diode drop and resistor values shift the stabilized amplitude.

Design Trade-Offs

  • Diode stabilization is simple and cheap, but it introduces some distortion because conduction happens mainly near the peaks.
  • Lamp or JFET stabilization can yield lower distortion, but is more complex or more temperature-sensitive in other ways.
  • Real products must consider resistor tolerances, diode temperature coefficient, op-amp slew rate, output swing, and bias-current effects.

From lab to product: the idealized theory says “gain 3 at resonance,” but real engineers add margin for startup, then use a nonlinear control element so the average effective gain relaxes back to 3. The amplitude is therefore set not just by linear RC theory, but by how the nonlinear stabilizer reshapes the feedback near the waveform peaks.

PART 1 — Problem Analysis (No Solving Yet)

Problem Restated Clearly

We must determine the steady-state output amplitude of a Wien-bridge oscillator after the variable feedback-setting potentiometer is replaced by a fixed divider: 15 kΩ from node b to the inverting input and 10 kΩ from the inverting input to ground. The nonlinear branch between nodes a and b contains a 10-kΩ resistor in parallel with antiparallel diodes, each modeled as a 0.65-V source in series with 100 Ω.

Category Information
Given Wien-bridge oscillator, equal RC network, attenuation 1/3 at oscillation frequency, fixed divider 15 kΩ and 10 kΩ, nonlinear branch 10 kΩ || diode pair, diode model 0.65 V + 100 Ω.
Unknown Peak-to-peak output amplitude at node b.
Need to find Vpp = 2 Vpeak.

Relevant Principles and Why They Apply

  • Ideal op-amp equality applies because the circuit is intended to operate in linear closed-loop sinusoidal steady state.
  • Wien-bridge gain of 1/3 applies because the oscillator is operating at its selected zero-phase frequency.
  • KCL at node b applies because node-current balance is the cleanest way to relate the divider current to the current through the nonlinear branch.
  • Piecewise diode conduction applies because at signal peaks one diode is forward biased.

Assumptions

  • The op-amp is ideal enough that v+=v- during steady oscillation.
  • The oscillation is approximately sinusoidal and occurs at the Wien-bridge resonant frequency.
  • The output amplitude is set by the waveform peaks, where one diode conducts.
  • Loading by the op-amp inputs is negligible.
  • Temperature dependence of the diode drop is ignored.

Possible Approaches

Approach A: Full nonlinear time-domain analysis

Pros: most exact in principle.

Cons: unnecessary, cumbersome, and not what textbook problems usually expect.

Approach B: Small-signal loop-gain analysis only

Pros: simple for startup.

Cons: cannot directly give the stabilized amplitude.

Approach C: Peak-condition piecewise-linear analysis

Pros: directly uses the diode model and yields the amplitude cleanly.

Cons: approximate, but standard and physically insightful.

Best choice: Approach C, because the problem explicitly gives a piecewise-linear diode model and asks for amplitude, not startup rate.

Meaning: at oscillation, the noninverting input sees one-third of node a.
Step 2 — Write the inverting-input divider relation.
Tool: voltage division across 15 kΩ and 10 kΩ.
Meaning: the inverting input is a fixed fraction of node b.
Step 3 — Enforce the ideal op-amp condition.
Tool: v+=v-.
Meaning: links node a and node b.
Step 4 — Define the branch voltage x = va-vb.
Tool: algebraic substitution.
Meaning: simplifies the nonlinear current law.
Step 5 — Write the current needed by the divider chain.
Tool: KCL at node b.
Meaning: the nonlinear branch must supply this current.

Common Mistakes

  • Using 25 kΩ as if it were directly between a and ground. It is connected through node b, so first relate a and b.
  • Forgetting the diode’s 0.65-V offset.
  • Using both diode branches simultaneously. Near a positive peak, one is on and the other is off.
  • Reporting peak amplitude when the question asks for peak-to-peak.

PART 3 — Full Solution (Detailed + Teaching)

Physical Intuition First

A Wien-bridge oscillator needs an amplifier gain of 3 at the selected frequency. Here the fixed divider on the inverting side sets a definite feedback ratio, while the diodes allow only a modest difference between the op-amp output node a and the external output node b. So the amplitude is determined by the point where the diode branch just supplies the current required by the divider load at the waveform peaks.

Step 1: Resonant relation of the Wien bridge

For the equal-R, equal-C Wien network, at the oscillation frequency ω = ω0, the transfer from node a to the noninverting input is real and equal to 1/3:

v+(t) = va(t)/3

This step is valid because at resonance the phase shift of the Wien network is zero, so the steady sinusoid passes with only attenuation.

Step 2: Inverting-input divider relation

The negative input is connected to node b through 15 kΩ and to ground through 10 kΩ. Since the op-amp input current is negligible, this is an ordinary divider:

v-(t) = [10 kΩ / (15 kΩ + 10 kΩ)] vb(t) = 0.4 vb(t)

Step 3: Ideal op-amp equality

In the linear operating regime of the closed loop,

v+(t) = v-(t)

Combining Steps 1 and 2:

va(t)/3 = 0.4 vb(t) va(t) = 1.2 vb(t)

Therefore the voltage across the nonlinear branch between a and b is

x(t) ≡ va(t) - vb(t) = 1.2 vb(t) - vb(t) = 0.2 vb(t)

Hence

vb(t) = 5x(t)

Step 4: Current required by the fixed divider

Because the op-amp input current is essentially zero, the current flowing from node b into the 15-kΩ and 10-kΩ chain is the same through both resistors. So the load seen from node b to ground is

15 kΩ + 10 kΩ = 25 kΩ

Thus the current that must be supplied by the branch between a and b is

i(t) = vb(t) / 25 kΩ

Since vb = 5x,

i(t) = (5x)/25 kΩ = x/5 kΩ

Step 5: Current through the nonlinear branch at a waveform peak

Near a positive peak, x = va-vb > 0, so one diode is forward biased and the other is reverse biased. The branch between a and b then consists of:

  • a 10-kΩ resistor carrying current x/10 kΩ, in parallel with
  • a conducting diode branch carrying current (x-0.65)/100 Ω, provided x > 0.65 V.

Therefore, at the peak:

i = x/10 kΩ + (x - 0.65)/100 Ω

Step 6: Match required current to actual current

The same current must satisfy both the divider requirement and the branch law, so

x/5 kΩ = x/10 kΩ + (x - 0.65)/100 Ω

Now solve carefully.

x/5000 - x/10000 = (x - 0.65)/100 x/10000 = (x - 0.65)/100

Multiply both sides by 10,000:

x = 100(x - 0.65)

Expand:

x = 100x - 65

Rearrange:

99x = 65

Hence

x = 65/99 ≈ 0.6566 V

Step 7: Convert to output peak and peak-to-peak amplitude

Since vb = 5x,

Vpeak = vb,peak = 5(0.6566) ≈ 3.283 V

Therefore,

Vpp = 2Vpeak ≈ 2(3.283) ≈ 6.57 V

Final Answer

Peak-to-peak output amplitude ≈ 6.57 V

So the output sinusoid is approximately 6.6 V peak-to-peak (more precisely, about 6.57 Vpp).

Sanity Checks

Units / Dimensions

Every term in the current equations has units of amperes:

V/Ω = A

The final answer is in volts, as required.

Physical Plausibility

The voltage across the nonlinear branch is only about 0.657 V at the peak, which is exactly the scale where a silicon diode begins to conduct strongly. That is physically consistent with amplitude stabilization by soft limiting.

Limiting-Case Interpretation

If the diode threshold were much larger, conduction would begin later and the output amplitude would be larger. If the diode’s series resistance were smaller, the limiter would be “harder” and the amplitude would settle closer to a fixed clipping level. Here the 100-Ω series resistance slightly softens the limit.

Optional deeper note: why no PDE operators like ∇, ∇·, or Laplacian appear here

This is a lumped-circuit problem. The spatial dimensions of the components are assumed small enough that voltages and currents can be represented by node variables and branch currents rather than continuous fields. In that regime, Maxwell’s equations reduce to circuit laws after suitable lumping assumptions. So the natural mathematical language is algebraic and ODE-based circuit theory, not field PDEs.

PART 4 — Deeper Understanding (Theory Around the Result)

What the Final Formula is Really Saying

The amplitude is set by a balance between two facts:

  1. The fixed divider forces a specific ratio between va and vb.
  2. The diode branch only allows enough extra current when va-vb becomes roughly one diode drop.

That is why the final amplitude is tied to the diode threshold, scaled by the feedback geometry.

Parameter Dependence

Parameter If increased Effect on amplitude
Diode threshold (0.65 V) Conduction starts later Amplitude increases
Diode series resistance (100 Ω) Limiter becomes softer Amplitude increases slightly
15-kΩ resistor Changes divider ratio Changes relation between a and b, so amplitude shifts
10-kΩ resistor to ground Changes divider ratio Also changes required branch current and amplitude

Alternative Derivation Idea

Another way is to derive the effective feedback factor of the noninverting amplifier and impose that the effective gain at resonance be exactly 3. Then, using the piecewise branch law of the parallel resistor-diode network, one solves for the peak branch voltage. This is mathematically equivalent to the KCL method used above, but the node-current route is cleaner and more transparent.

Concept Check

  • Q: Why is the output not determined only by the Wien RC values?
    A: The RC values set the frequency; the amplitude is set by the nonlinear stabilization branch.
  • Q: Why can one diode be ignored at a positive peak?
    A: Because the antiparallel partner is reverse biased and therefore open in the piecewise model.
  • Q: Why do we use the peak condition?
    A: Because that is where diode conduction first becomes significant and clamps further growth.
  • Q: Why is the result only approximate?
    A: The actual waveform is not a mathematically perfect sinusoid once the diodes conduct, and real diodes are not exactly piecewise linear.
