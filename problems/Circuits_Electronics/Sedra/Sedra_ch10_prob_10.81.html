10.81 Refer to Fig. 10.31(c). In situations in which Rsig is large, the high-frequency response of the source follower is determined by the low-pass circuit formed by Rsig and the input capacitance. An estimate of Cin can be obtained by using the Miller approximation to replace Cgs with an input capacitance C eq = Cgs(1−K) where K is the gain from gate to source. Using the low-frequency value of K = gmRL′ /(1 + gmRL′ ) find C eq and hence Cin and an estimate of fH.
