Three Identical Common-Source Stages in a Loop: Oscillation Condition, Frequency, and Minimum gm

We analyze a three-stage loop oscillator built from identical common-source amplifier stages, each modeled as a first-order inverting amplifier with output resistance R and output capacitance C. The task is to apply the Barkhausen conditions carefully and derive the required per-stage phase shift, the oscillation frequency ω0, and the minimum transistor transconductance required for sustained oscillation.

Symbolic result First-order small-signal model Barkhausen phase + magnitude Design-oriented interpretation

Quick Summary

  • This problem is about a three-stage loop oscillator made from identical common-source amplifier stages.
  • The key physics and circuit idea is the feedback oscillation condition: around the loop, the total phase shift must be an integer multiple of 360° and the loop-gain magnitude must be at least unity.
  • Each stage is modeled by the transfer function A(s) = -gmR/(1+sRC), which is an inverting first-order low-pass amplifier.
  • The governing equations are the Barkhausen conditions: ∠L(jω0) = -360° k and |L(jω0)| = 1, where L(s) = [A(s)]3.
  • Because each stage already contributes an inversion, the extra pole phase lag per stage must be 60°, so the total phase of each stage is -240°, equivalently +120° modulo 360°.
  • The oscillation frequency is ω0 = √3/(RC).
  • At that frequency, each stage has magnitude |A(jω0)| = gmR/2.
  • For sustained oscillation, the minimum required transconductance is gm,min = 2/R.
  • The result is symbolic, and physically it says the transistor must be strong enough to compensate for the attenuation caused by the stage pole at the oscillation frequency.

PART 0 — Concept Primer (Theory Before Solving)

Core definitions

Symbol Meaning SI Unit
gm Small-signal transistor transconductance; output current per input gate-source voltage S = A/V
R Effective output resistance seen at the stage output Ω
C Effective output capacitance at the stage output F
s Laplace variable, s = σ + jω s-1
ω Angular frequency rad/s
A(s) Transfer function of one stage dimensionless
L(s) Loop gain dimensionless

Physical meaning of the key quantities

Transconductance gm tells us how effectively the transistor converts input voltage variations into output current variations.

Output resistance R and output capacitance C together create a first-order time constant RC, which introduces both attenuation and phase lag.

The oscillator works only when the loop can feed back a signal with the right phase and with enough magnitude to sustain itself.

Key laws and principles

1) Small-signal linear model: We linearize each transistor around its bias point. That makes the circuit analyzable using transfer functions and phasors.

2) First-order output node dynamics: The drain/output node behaves like an RC load, so one stage behaves as a one-pole low-pass amplifier.

3) Barkhausen criterion: At the oscillation frequency ω0, the total loop phase must be a multiple of 360° and the loop magnitude must be unity for sustained oscillation.

Mathematical model

State variable: Let vo(t) be the output voltage of one stage.

Domain: Lumped-circuit, time domain t ≥ 0, then transformed to frequency domain via Laplace/phasor analysis.

Coordinate system: Not spatial; this is a lumped one-node dynamical system, so ordinary differential equations are sufficient. No field operators such as ∇, ∇·, or ∇× are needed because the system is not spatially distributed in this model.

Node equation for one stage: using KCL at the output node,

One-stage small-signal ODE C (dv_o/dt) + (v_o/R) = -g_m v_i

This is valid because the stage is assumed linearized, time-invariant, and first-order, with all parasitics absorbed into the effective R and C.

Taking the Laplace transform with zero initial condition,

Transfer function (C s + 1/R) V_o(s) = -g_m V_i(s) A(s) = V_o(s)/V_i(s) = -g_m / (C s + 1/R) = -g_m R / (1 + s R C)

Boundary/initial condition used for transfer-function derivation: zero initial stored capacitor voltage in the linear small-signal response, which is the standard condition for transfer functions.

Why this mathematical form matters

This single-stage equation is a linear first-order ODE. In the frequency domain, it becomes a simple rational function. The oscillator is then a question of whether the product of three identical complex gains can close the loop with the right angle and magnitude.

Math Insight

The core equation A(s) = -gmR/(1+sRC) is a one-pole complex transfer function. Mathematically, it is a stable first-order linear system with a pole at s = -1/(RC). Oscillation does not come from an unstable single stage; it comes from the closed-loop phase closure of three such stages connected in a loop.

Common models and approximations used here

  • Small-signal approximation: transistor current is linearized about a bias point, so i_d ≈ g_m v_{gs}.
  • Single-pole approximation: all output-node nonidealities are lumped into one effective R and one effective C.
  • Identical stages: same g_m, R, and C for each stage.
  • Frequency-domain steady-state analysis: we evaluate the response at s = jω because oscillation is sinusoidal in steady state.

Mini intuition examples

Example 1: Low frequency

If ωRC ≪ 1, then the pole adds almost no lag, so each stage is close to an ideal inverter: A(jω) ≈ -gmR.

Example 2: At the pole scale

If ωRC = 1, then each stage adds a pole lag of 45°, so its total phase is -225°.

Example 3: High frequency

If ωRC ≫ 1, each stage approaches a phase near -270° and its magnitude falls as 1/ω.

What to watch for

  • Do not forget the inversion sign of the common-source stage.
  • Do not set the total loop phase to -360° without checking whether the single-stage phase can actually reach the required value.
  • Do not confuse the phase of the pole alone with the phase of the entire stage.
  • Do not impose the magnitude condition before evaluating it at the correct oscillation frequency.
  • For “minimum gm required,” use the equality case of the magnitude condition.

PART 0B — Industry & Applied Physics Perspective

Where this phenomenon appears in industry

  • Clock generation in digital ICs: multi-stage delay-based oscillators are widely used in on-chip timing, built-in self-test, and low-area clocking blocks.
  • Sensor interface electronics: oscillators translate capacitance or resistance changes into frequency shifts for MEMS, touch sensing, and chemical/biomedical sensors.
  • PLL/VCO subsystems: transistor-stage oscillators and their variants appear in frequency synthesis and communication integrated circuits.
  • Process-monitor circuits: ring oscillators are used to characterize manufacturing variation, temperature drift, and transistor aging in semiconductor fabrication.

Why engineers care

Performance

Oscillation frequency sets timing speed, bandwidth, and synchronization capability. Designers care about startup margin, frequency accuracy, phase noise, and power dissipation.

Reliability and cost

If gm is too small, oscillation may fail to start. If it is too large, power and nonlinear distortion increase. A robust but not wasteful margin is crucial for manufacturability and yield.

Typical measurement and validation methods

  • Frequency spectrum measurement with a spectrum analyzer or oscilloscope FFT.
  • Startup verification in transient simulation and lab measurement.
  • Small-signal AC loop analysis in design tools to estimate phase and loop gain.
  • PVT sweeps: process, voltage, and temperature validation to ensure the oscillator works across all corners.

Design trade-offs and constraints

  • Larger capacitance C improves noise filtering but lowers oscillation frequency.
  • Larger resistance R can raise voltage gain for a given gm, but may slow the circuit.
  • Larger gm helps startup but costs bias current and power.
  • Parasitics and mismatch shift the actual frequency away from the ideal symbolic result.

From lab to product

The idealized model says ω0 = √3/(RC) and gm,min = 2/R. In a real IC, engineers replace R and C by effective, bias-dependent, process-dependent values. They then add startup margin, account for parasitic coupling, and use simulation plus silicon characterization to calibrate the real oscillator against the ideal theory.

PART 1 — Problem Analysis (No Solving Yet)

Problem restated clearly

We have three identical common-source amplifier stages connected in a closed loop. Each stage behaves like an inverting amplifier with a one-pole output network characterized by an effective resistance R and capacitance C. We must determine:

  1. the phase angle each amplifier stage must provide for oscillation,
  2. the oscillation frequency ω0, and
  3. the minimum gm of each transistor required for sustained oscillation.

Given quantities

  • Three identical stages
  • Each stage is a common-source amplifier
  • Effective output parameters: R and C
  • We are to assume the small-signal model that includes output resistance and capacitance

Unknowns

  • Per-stage phase angle at oscillation
  • Oscillation frequency ω0
  • Minimum gm for sustained oscillation

Relevant principles and why they apply

Principle Why it applies Why alternatives are less central
Barkhausen criterion The question explicitly asks for oscillation conditions in a looped amplifier system. Energy methods alone would not directly give phase and frequency conditions.
Small-signal transfer function Each stage is linearized around bias and represented by -gmR/(1+sRC). Full nonlinear device equations are unnecessary for the symbolic oscillation condition.
Phasor/frequency-domain analysis Oscillation frequency is determined from phase closure at s = jω. Pure time-domain transient analysis is less efficient for this symbolic derivation.

Assumptions

  • Each stage is identical and independently modeled by one pole.
  • Bias circuitry does not affect the small-signal loop transfer significantly.
  • We neglect additional poles, zeros, body effect, and frequency dependence of gm.
  • The oscillation analysis is done at steady-state frequency using linear phasors.
  • For the minimum gm, we use the equality case of the magnitude condition.

Possible approaches

Approach A: Direct transfer function

Write one-stage A(s), then form L(s)=A(s)3, and apply Barkhausen. Best for this problem.

Approach B: Time-domain ODE chain

Write three coupled first-order equations and search for sinusoidal eigenmodes. More formal, but heavier algebra.

Approach C: Nyquist perspective

Interpret the loop transfer in the complex plane. Useful conceptually, but unnecessary for the symbolic answer asked here.

Chosen approach

We choose Approach A because the circuit is explicitly described in terms of identical amplifier stages with effective R and C, so the one-stage transfer function is immediate and the Barkhausen conditions produce the answer with maximum clarity and minimum unnecessary complexity.

PART 2 — Strategy & Tips (Roadmap Only)

  1. Model one stage.
    Tool: KCL + Laplace transform.
    Meaning: obtain the stage transfer function that encodes gain and phase.
  2. Build the loop gain.
    Tool: identical-stage multiplication, L(s)=A(s)3.
    Meaning: total effect after one full trip around the loop.
  3. Apply the phase condition.
    Tool: argument of a complex number.
    Meaning: find the frequency at which the fed-back signal reinforces itself.
  4. Solve for ω0.
    Tool: inverse tangent relation.
    Meaning: convert the phase requirement into the oscillation frequency.
  5. Evaluate the magnitude at that frequency.
    Tool: magnitude of 1/(1+jx).
    Meaning: determine how much each stage attenuates/amplifies at oscillation.
  6. Apply the unity loop-gain condition.
    Tool: |L(jω0)|=1.
    Meaning: find the minimum gm needed for sustained oscillation.
  7. Check dimensions and physical plausibility.
    Tool: dimensional analysis and limiting cases.
    Meaning: verify the formula is mathematically and physically sensible.

Common mistakes

  • Using only the pole phase -tan-1(ωRC) and forgetting the inversion angle -180°.
  • Choosing the wrong multiple of 360° in the phase condition.
  • Writing gm,min = 1/R by ignoring attenuation from the pole.
  • Skipping the step of evaluating the magnitude specifically at the oscillation frequency.

PART 3 — Full Solution (Detailed + Teaching)

Physical intuition before calculating

A common-source stage inverts the signal, so one stage contributes roughly 180° of phase reversal, plus some extra lag from its output pole. Three such stages therefore provide more than 540° total phase lag. Oscillation occurs only at the frequency where this total lag becomes equivalent to an integer multiple of 360°. At that same frequency, the loop must have exactly unity magnitude for sustained sinusoidal oscillation.

Step 1: Derive the transfer function of one stage

Let v_i(t) be the input small-signal voltage of one stage and v_o(t) its output. The transistor produces a small-signal current approximately equal to gm v_i. Because it is a common-source stage, the output is inverted; therefore the controlled current contributes with a negative sign in the voltage transfer relation.

Applying KCL at the output node gives

KCL at the output node C (dv_o/dt) + v_o/R = -g_m v_i

This is a linear first-order ODE. The validity comes from the small-signal linearization and the lumped RC approximation for the output node.

Taking the Laplace transform with zero initial conditions,

Laplace-domain relation C s V_o(s) + V_o(s)/R = -g_m V_i(s)

Factor out V_o(s):

Factorization V_o(s) [C s + 1/R] = -g_m V_i(s)

Thus the one-stage transfer function is

One-stage transfer function A(s) = V_o(s)/V_i(s) = -g_m / (C s + 1/R) = -g_m R / (1 + sRC)

Step 2: Write the loop gain

Since the three stages are identical and connected in a loop, the loop gain is

Loop gain L(s) = [A(s)]^3 = [-g_m R / (1 + sRC)]^3

To study oscillation at steady state, set s = jω:

Loop gain on the imaginary axis L(jω) = [-g_m R / (1 + jωRC)]^3

Step 3: Extract the phase of one stage

For one stage,

Single-stage phase ∠A(jω) = ∠(-g_m R) - ∠(1 + jωRC) = -180° - tan^(-1)(ωRC)

Here:

  • -gmR contributes -180° because of inversion.
  • 1 + jωRC has phase +tan-1(ωRC) in the denominator, so it contributes a lag of -tan-1(ωRC).

Step 4: Apply the Barkhausen phase condition

The total loop phase is three times the phase of one stage:

Total loop phase ∠L(jω) = 3[-180° - tan^(-1)(ωRC)] = -540° - 3 tan^(-1)(ωRC)

For oscillation, the loop phase must be an integer multiple of -360°. Since tan-1(ωRC) > 0 for positive frequency, the quantity -540° - 3 tan-1(ωRC) is more negative than -540°. Therefore the physically correct multiple is -720°, not -360°.

Phase condition at oscillation -540° - 3 tan^(-1)(ω_0 RC) = -720°

Solve step by step:

Algebraic solution for phase 3 tan^(-1)(ω_0 RC) = 180° tan^(-1)(ω_0 RC) = 60° ω_0 RC = tan 60° = √3

Hence,

Oscillation frequency ω_0 = √3 / (RC)

What phase angle does each stage provide?
At oscillation, the pole contributes -60°. Therefore each stage contributes

Per-stage phase angle ∠A(jω_0) = -180° - 60° = -240°

Since phase is defined modulo 360°, this is equivalent to +120°. Many textbook answers state that each stage must provide 120° of phase shift around the loop; in strict transfer-function angle notation, the stage angle is -240°.

Step 5: Apply the magnitude condition

The magnitude of one stage is

Single-stage magnitude |A(jω)| = g_m R / |1 + jωRC| = g_m R / √(1 + (ωRC)^2)

At the oscillation frequency, ω0RC = √3, so

Magnitude at oscillation |A(jω_0)| = g_m R / √(1 + 3) = g_m R / 2

Therefore the loop magnitude is

Loop magnitude at oscillation |L(jω_0)| = (g_m R / 2)^3

For sustained oscillation, the minimum condition is equality:

Unity magnitude condition (g_m R / 2)^3 = 1

Take the cube root:

Minimum transconductance g_m R / 2 = 1 g_m,min = 2 / R

Final boxed result

Per-stage phase angle at oscillation: each stage must provide 120° of net loop-compatible phase shift, which in transfer-function angle notation is -240° per stage.

Oscillation frequency:

Final symbolic result ω_0 = √3 / (RC)

Minimum transistor transconductance for sustained oscillation:

Final symbolic result g_m,min = 2 / R

Sanity checks

1) Dimensional analysis

RC has units of seconds, so 1/(RC) has units s-1, consistent with angular frequency. Also, 2/R has units 1/Ω = S, consistent with gm.

2) Limiting-case reasoning

If C increases, the node slows down, so ω0 should decrease. Our formula gives exactly that. If R increases, the stage gain for a given gm becomes larger, so the required minimum gm should fall. Our formula gives gm,min = 2/R, so it does.

Small-parameter and large-parameter interpretation

Introduce the nondimensional variable x = ωRC. Then the phase condition becomes 3[-180° - tan-1(x)] = -720°, so the answer depends only on the dimensionless combination x. This is a simple scaling result: the physics is controlled by the ratio of oscillation timescale to the output-node time constant.

At low x, tan-1(x) ≈ x, so the extra phase lag is too small. At very large x, the magnitude collapses as 1/x. Thus oscillation happens at an intermediate scaled frequency, here exactly x = √3.

Optional deeper note: an alternative ODE/eigenmode viewpoint

You could write three coupled first-order equations, one for each stage output, and look for solutions of the form v_k(t) = Re{V_k e^{jωt}}. This turns the problem into a complex eigenvalue condition for the three-stage cyclic system. Because the stages are identical, the loop-product method used above is mathematically equivalent and much simpler.

PART 4 — Deeper Understanding (Theory Around the Result)

What the final formulas mean physically

  • ω0 = √3/(RC): the oscillation frequency is set entirely by the stage time constant. Faster stages oscillate faster.
  • gm,min = 2/R: the transistor must generate enough small-signal current to compensate the per-stage attenuation at the oscillation frequency.
  • Per-stage phase = -240° (or +120° equivalent): each stage is not just an inverter; its pole must also add exactly 60° of extra lag.

Parameter dependence

Parameter Effect on ω0 Effect on startup/sustained oscillation
R ω0 ∝ 1/R Larger R lowers required gm since gm,min=2/R
C ω0 ∝ 1/C Does not directly enter gm,min in this ideal model, but it shifts frequency and therefore practical parasitic effects
gm No direct effect on ideal ω0 Higher gm increases loop magnitude and startup margin

Alternative derivation idea

A compact alternative is to write the one-stage phase as π + tan-1(ωRC) in radians modulo 2π, then impose 3(π + tan-1(ωRC)) = 4π. This immediately gives tan-1(ωRC)=π/3, hence ωRC=√3.

Concept check Q&A

  • Q: Why is the correct total phase multiple -720° instead of -360°?
    A: Because each stage already contributes at least -180°, so three stages contribute at least -540°. The next valid multiple is -720°.
  • Q: Why does C affect frequency but not the final symbolic gm,min?
    A: In this one-pole identical-stage model, once the phase condition fixes ωRC=√3, the attenuation becomes exactly a factor of 2 per stage, independent of the separate values of R and C.
  • Q: What happens if gm < 2/R?
    A: The loop gain is below unity at the phase-closure frequency, so oscillations decay instead of sustaining.
  • Q: What happens if gm > 2/R?
    A: The oscillation starts and grows until nonlinearities reduce the effective loop gain to unity.
