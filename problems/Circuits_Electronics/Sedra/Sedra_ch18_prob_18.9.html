Topic: op-amp comparator obtained from a limiter by removing the linear feedback resistor.

Electronics / Applied Physics

Comparator Design

Limiter Levels

Designing the Comparator Characteristic from Fig. 18.4(a)

We convert the limiter into a comparator by removing Rf, derive the output levels from the diode-resistor ladder, prove how a bias source shifts the switching threshold, and then choose standard 5% resistor values to realize output levels of ±3 V and a threshold at vI = +2 V.

Quick Summary

  • This problem is about an op-amp comparator obtained by removing the linear feedback resistor from a diode limiter circuit.
  • The key physical idea is that the op-amp saturates, while the diode-resistor ladder fixes the output plateaus at finite positive and negative levels.
  • The governing relations come from ideal-op-amp logic, KCL, and piecewise-linear circuit analysis.
  • With ideal diodes and rails ±V, the output levels are L+ = (R4/R5)V and L- = -(R3/R2)V.
  • Adding a dc source VB through RB shifts the threshold to vI,th = -(R1/RB)VB.
  • For the required design, choose R1 = 100 kΩ, R2 = R5 = 30 kΩ, R3 = R4 = 18 kΩ, and RB = 250 kΩ with VB = -5 V.
  • The result is a numeric design: output levels ±3 V, threshold +2 V, input resistance 100 kΩ, and plateau slope 0 V/V, which certainly satisfies the ≤0.05 V/V requirement.
Key equations
L+ = (R4/R5) V
L- = -(R3/R2) V
vI,th = -(R1/RB) VB
For ±3 V with V = 5 V: R4/R5 = R3/R2 = 3/5 = 0.6
For vI,th = +2 V with R1 = 100 kΩ and VB = -5 V: RB = 250 kΩ

PART 0 — Concept Primer

Core definitions

  • vI (V): input voltage applied through R1.
  • vO (V): op-amp output voltage.
  • +V, -V (V): dc supply rails for the resistor ladder.
  • R1 (Ω): input resistor, sets input resistance seen by the source.
  • R2, R3, R4, R5 (Ω): ladder resistors that determine the plateau values.
  • D1, D2: ideal diodes used as state-selecting feedback elements.
  • RB, VB: threshold-shifting network connected to the inverting-input node.

Physical meaning

  • The op-amp compares the inverting-input node voltage with ground at the non-inverting terminal.
  • Without Rf, the circuit is no longer a linear amplifier; it becomes a piecewise nonlinear switching device.
  • The ladder plus diodes do not merely “clip”; they define reproducible output states.
  • The added bias source shifts the switching point horizontally along the input axis, which is the same idea used in threshold detectors and level-crossing circuits.
Key principles and validity conditions
  • Ideal op-amp approximation: infinite open-loop gain, zero input current, and output able to move as needed until constrained by the circuit state.
  • Ideal diode approximation: when ON, the diode drop is 0 V; when OFF, the current is 0.
  • Piecewise-linear analysis: we solve the circuit separately in the positive-output state and the negative-output state, then stitch the results together.
  • Steady-state dc analysis: no capacitors or inductors are involved, so no differential equation in time is needed here.

Math Insight

The core equation here is not a PDE or wave equation; it is a piecewise algebraic switching law. Mathematically, the circuit is a hybrid system: each diode configuration defines a linear algebraic subproblem, while the diode ON/OFF conditions define the state boundaries. In other words, the comparator characteristic is a piecewise-constant map from input voltage to output voltage, shifted by a linear bias relation.

Mathematical model

Let x denote the inverting-input node voltage. The non-inverting input is grounded, so v+ = 0. The op-amp state is determined by the sign of v+ - v- = -x.

The model is static and piecewise:

State law
If x > 0, then vO goes negative and the lower diode path becomes relevant.
If x < 0, then vO goes positive and the upper/lower conduction state switches accordingly.
At the switching boundary, x = 0.

Because the op-amp input current is zero, KCL at node x gives, before switching,

Threshold relation
x = weighted node voltage determined by sources through R1 and RB
At x = 0: vI/R1 + VB/RB = 0
Therefore vI,th = -(R1/RB) VB

This is dimensionally correct because R1/RB is dimensionless, so the right-hand side has units of volts.

Mini intuition examples

Example 1

If no bias source is attached, symmetry suggests the switching threshold is at 0 V. That is exactly what the formula gives when VB=0.

Example 2

If VB is negative, then -(R1/RB)VB is positive, so the whole transfer curve shifts to the right.

Example 3

If R4/R5 increases, the positive output plateau rises. This is a pure resistor-ratio effect.

What to watch for
  • Do not confuse the op-amp supply rails with the defined comparator output plateaus; the ladder sets the latter.
  • Be careful with sign conventions: a positive threshold shift to +2 V requires a negative bias source if all resistors are positive.
  • The input resistance requirement refers to what the source sees at vI, which is set by R1.
  • When Rf is removed, the output plateaus are flat, so the slope in those regions is zero.

PART 0B — Industry & Applied Physics Perspective

Comparator and limiter ideas appear everywhere in engineering because real systems often need to decide whether a signal is above or below a threshold, while keeping outputs within safe, standardized voltage ranges.

Where this appears in industry

  • Industrial automation: sensor thresholding, proximity detection, over/under-voltage alarms.
  • Power electronics: zero-crossing detection, current-limit supervision, fault detection.
  • Instrumentation: window comparators, level detectors, trigger conditioning for ADCs and counters.
  • Medical devices: physiological signal threshold triggers and safety interlocks.
  • Telecom and digital interfaces: analog-to-digital front ends and pulse restoration.

Why engineers care

  • Performance: reliable switching at the desired threshold.
  • Safety: fixed output levels prevent downstream overdrive.
  • Reliability: diode-limited levels can be more predictable than unconstrained op-amp saturation.
  • Cost: resistor-ratio designs are cheap and easy to manufacture.
  • Manufacturability: standard resistor values and tolerances matter in mass production.

Measurement and validation in practice

  • Apply a slow ramp at the input and observe the switching point on an oscilloscope.
  • Measure output high and low plateaus with a DMM or scope.
  • Check tolerance spread by Monte Carlo simulation or worst-case analysis.
  • Validate diode non-ideality and op-amp saturation headroom in the lab.

Design trade-offs and real constraints

  • Real diodes have nonzero forward drop and temperature dependence.
  • Real op-amps have finite slew rate, offset voltage, input bias current, and limited output swing.
  • Higher resistor values reduce current draw but increase sensitivity to leakage and noise.
  • Tighter threshold accuracy may require 1% resistors or trimming, not just 5% parts.
From lab to product

The idealized textbook model is excellent for first design, but product-grade circuits usually add hysteresis, account for diode drops, and use tolerance analysis. In industrial environments, noise can cause chatter near the threshold; a practical design often modifies the circuit into a Schmitt trigger or uses a dedicated comparator IC.

PART 1 — Problem Analysis

Problem restated

We start from the limiter circuit of Fig. 18.4(a), but the resistor Rf is removed. This turns the circuit into a comparator-like device with two finite output levels. We must:

  1. Sketch the transfer characteristic.
  2. Show that adding a dc source VB to the virtual-ground node through RB shifts the characteristic horizontally to vI = -(R1/RB)VB.
  3. Choose standard 5% resistor values so the output levels are ±3 V, the threshold is +2 V, the input resistance is 100 kΩ, and the limiting-region slope is ≤0.05 V/V.

Given quantities

  • Supply rails for the ladder: +V = +5 V, -V = -5 V
  • Available bias source: ±5 V
  • Desired output plateaus: +3 V and -3 V
  • Desired comparator threshold: vI,th = +2 V
  • Desired input resistance: 100 kΩ
  • Ideal diodes: VD = 0

Unknowns

  • Transfer characteristic vO(vI)
  • Resistor values R1, R2, R3, R4, R5, RB
  • Bias source polarity and value used for VB

Relevant principles and why they apply

  • KCL: needed at the inverting-input node and along the resistor ladder.
  • Ideal-op-amp switching logic: needed because removing Rf destroys linear feedback over the whole range.
  • Ideal-diode state selection: needed because only one clamp path conducts in each output state.
  • Resistor-divider relations: needed to determine the output plateaus.
Assumptions
  • Op-amp is ideal for the purpose of the transfer characteristic.
  • Diodes are ideal: ON → short circuit, OFF → open circuit.
  • The analysis is steady-state dc.
  • Rf is removed, so the plateaus are flat and the slope in those regions is zero.

Possible approaches

Approach Idea Pros Cons
State-by-state diode analysis Assume one diode ON at a time and solve each linear subcircuit. Direct, physically transparent, ideal for design. Requires careful sign tracking.
Thevenin-equivalent viewpoint Reduce the ladder to equivalent sources seen from nodes A and B. Elegant for advanced readers. Less intuitive for beginners.
Numerical nodal matrix method Write linear equations for each switching state. Systematic and scalable. Overkill for this small circuit.

Best choice: the state-by-state diode analysis. It exposes the physics clearly, produces the transfer characteristic immediately, and leads naturally to component selection.

PART 2 — Strategy & Tips

  1. Identify the switching condition. Use the inverting-input node voltage x and set the threshold at x = 0.
  2. Derive the threshold shift. Apply KCL at node x with sources vI and VB through resistors R1 and RB.
  3. Find the negative plateau. Assume the upper diode path conducts, set node A to virtual ground, and solve the small resistor-divider relation between +V, A, and vO.
  4. Find the positive plateau. Assume the lower diode path conducts, set node B to virtual ground, and solve the analogous divider relation.
  5. Apply the design targets. Set the resistor ratios for ±3 V using ±5 V rails.
  6. Use standard 5% values. Pick exact E24 values when possible.
  7. Check input resistance and slope. Verify R1 = 100 kΩ and note that with Rf removed the plateau slope is zero.
Common mistakes
  • Using the wrong sign for the threshold shift.
  • Setting the plateau equal to ±5 V instead of the resistor-ratio value.
  • Forgetting that R1 sets the input resistance.
  • Ignoring that ideal 5% resistor values can still be chosen to satisfy an exact ratio, even though actual manufactured parts have tolerance spread.

PART 3 — Full Solution

Physical intuition first

Once Rf is removed, the circuit is no longer a linear limiter with sloped outer regions. Instead, it behaves like a comparator: the output rapidly jumps between two nearly constant values. Those values are not arbitrary; they are set by the resistor ladder and whichever diode becomes forward biased. So before doing any algebra, we already expect a step-like transfer characteristic:

  • for input below threshold → one constant output level;
  • for input above threshold → the opposite constant output level.

Step 1: threshold without and with bias

Let the inverting-input node be x. The non-inverting input is grounded, so the switching boundary occurs when

Switching condition
x = 0

With the added bias source VB connected to node x through RB, and with zero op-amp input current, KCL at node x gives

KCL at the inverting-input node
(x - vI)/R1 + (x - VB)/RB = 0

At the threshold, x = 0. Substituting:

Threshold derivation
(0 - vI)/R1 + (0 - VB)/RB = 0
-vI/R1 - VB/RB = 0
vI = -(R1/RB) VB

Therefore the transfer characteristic is shifted along the input axis so that the switching threshold is

Threshold shift result
vI,th = -(R1/RB) VB

This is exactly what the problem asked us to show.

Why this step is mathematically valid

KCL is valid because the node is purely resistive and no charge is accumulating in steady state. The op-amp input current is zero in the ideal model, so only the two resistor currents appear. The equation is linear because the constitutive law for each resistor is linear: i = (v1-v2)/R.

Step 2: derive the output plateaus

We now analyze the two output states separately.

Negative output state

In the negative-output state, the upper diode path conducts and node A is effectively tied to the inverting-input virtual-ground level, which is 0 V in the switching model. Thus,

Upper clamp condition
vA = 0

The current through R2 from +V to A equals the current through R3 from A to vO:

Negative plateau derivation
(V - 0)/R2 = (0 - vO)/R3
V/R2 = -vO/R3
vO = -(R3/R2) V

Therefore,

Negative plateau
L- = -(R3/R2) V

Positive output state

In the positive-output state, the lower diode path conducts and node B is clamped to 0 V:

Lower clamp condition
vB = 0

Now the current through R5 from B to -V equals the current through R4 from vO to B:

Positive plateau derivation
(0 - (-V))/R5 = (vO - 0)/R4
V/R5 = vO/R4
vO = (R4/R5) V

Therefore,

Positive plateau
L+ = (R4/R5) V

Step 3: choose resistor ratios for ±3 V output levels

The supplies are ±V = ±5 V, and we want

Design targets for the plateaus
L+ = +3 V
L- = -3 V

Using the formulas above:

Resistor-ratio conditions
(R4/R5) · 5 = 3 → R4/R5 = 3/5 = 0.6
(R3/R2) · 5 = 3 → R3/R2 = 3/5 = 0.6

A convenient exact E24 choice is

Standard 5% plateau-setting values
R2 = 30 kΩ, R3 = 18 kΩ
R5 = 30 kΩ, R4 = 18 kΩ

because 18/30 = 0.6 exactly as a nominal value.

Step 4: set the input resistance

The source sees the circuit through R1, so to make the input resistance 100 kΩ, choose

Input resistance requirement
R1 = 100 kΩ

Step 5: shift the threshold to +2 V

We want

Desired threshold
vI,th = +2 V

Using the threshold formula,

Threshold design equation
2 = -(100 kΩ / RB) VB

Since all resistors are positive, the threshold becomes positive only if VB is negative. Using the available supply, choose

Choose bias polarity
VB = -5 V

Then

Solve for RB
2 = -(100 kΩ / RB)(-5)
2 = 500 kΩ / RB
RB = 250 kΩ

The nominal value 250 kΩ is easily realized using standard 5% resistors in series, for example

Practical 5% realization
RB = 100 kΩ + 150 kΩ = 250 kΩ

Step 6: transfer characteristic

With the chosen values, the comparator switches at vI = +2 V. Therefore the transfer characteristic is

Final transfer characteristic
vO = +3 V, for vI < +2 V
vO = -3 V, for vI > +2 V

At exactly vI = +2 V, the ideal model is at the switching boundary, so the output is mathematically indeterminate in the idealized infinitely fast limit; physically, noise and device imperfections pick one state.

Step 7: slope requirement

Since Rf is removed, the outer regions are horizontal plateaus:

Slope in limiting regions
dvO/dvI = 0

Therefore,

Check
0 ≤ 0.05 V/V

so the specification is automatically satisfied.

Sanity checks

Units

  • R4/R5 and R3/R2 are dimensionless.
  • Multiplying a dimensionless ratio by V gives volts, as required for output levels.
  • -(R1/RB)VB also has units of volts.

Limiting cases

  • If VB=0, then the threshold returns to 0 V.
  • If R4=R5, the positive plateau becomes +5 V.
  • If R3=R2, the negative plateau becomes −5 V.
Final Answer

Threshold shift:

vI,th = -(R1/RB) VB

Output levels with ideal diodes:

L+ = (R4/R5) V,    L- = -(R3/R2) V

Choose the following nominal 5% resistor values:

  • R1 = 100 kΩ
  • R2 = 30 kΩ
  • R3 = 18 kΩ
  • R4 = 18 kΩ
  • R5 = 30 kΩ
  • RB = 250 kΩ = 100 kΩ + 150 kΩ
  • VB = -5 V

Resulting characteristic:

vO = +3 V for vI < +2 V

vO = -3 V for vI > +2 V

Checks:

  • Input resistance = 100 kΩ
  • Limiting-region slope = 0 V/V ≤ 0.05 V/V

PART 4 — Deeper Understanding

Re-interpret the final formulas

R3/R2

Controls how far below zero the negative output plateau lies. Larger ratio → more negative output magnitude.

R4/R5

Controls the positive output plateau. Larger ratio → larger positive level.

R1/RB

Controls how strongly the bias source shifts the threshold. Larger ratio → larger shift for a given VB.

Parameter dependence

  • If V increases while the resistor ratios stay fixed, the plateau magnitudes scale linearly with V.
  • If VB becomes more negative, the threshold moves further to the right.
  • If R1 increases with RB fixed, the same bias source produces a larger threshold shift.
  • If real resistor tolerances are included, both threshold and output levels vary proportionally to those resistor-ratio errors.
Alternative derivation idea

One can replace the upper part of the ladder by a Thevenin equivalent seen from node A, and the lower part by another Thevenin equivalent seen from node B. Then each conducting diode simply clamps the inverting node to the Thevenin terminal voltage, making the plateau equations emerge from one-line divider formulas.

Concept checks

  • Q: Why is the characteristic shifted horizontally rather than vertically by VB? A: Because VB changes the condition at which the input node reaches 0 V, i.e. the switching input value.
  • Q: Why is the transfer characteristic inverting? A: The signal is applied to the inverting input while the non-inverting input is at ground.
  • Q: Why is the plateau not ±5 V? A: The diode-resistor ladder clamps the output to resistor-ratio-defined values before the full rail value matters at the output node.
  • Q: Why is the plateau slope zero here? A: Because the linear feedback resistor Rf was removed; the circuit is no longer a linear limiter in those regions.
